## 一、六大模块

### 1. 流量运营 — "用户来不来、来了买不买"
| 指标类型 | 核心指标 |
|---------|---------|
| 规模 | 闪购DAU（剔除外投）、美团App/外卖App DAU |
| 渗透 | 闪购DAU占美团App DAU占比 |
| 效率 | CTR（点击/曝光）、CXR（消费用户/设备） |

### 2. 用户增长 — "拉得来、留得住、花得起"
| 指标类型 | 核心指标 |
|---------|---------|
| 规模 | 日均交易用户数、日均新客数 |
| 结构 | 90后/00后交易用户占比 |
| 质量 | 新客次7日复购率、365新客次7日留存率 |
| 成本 | 新客全链路CAC、补贴CAC、渠道CAC |

### 3. 域外流量 — "出去找用户"
| 指标类型 | 核心指标 |
|---------|---------|
| 规模 | 域外交易用户数、域外新客数 |
| 结构 | 域外交易用户占比、域外新客贡献度 |
| 效率 | 域外新客CAC、域外新客次7日复购率 |

### 4. 权益运营（神券） — "花钱的效率"
| 指标类型 | 核心指标 |
|---------|---------|
| 规模 | 神券日均消费GTV、日均订单量 |
| 渗透 | 神券订单渗透率、主券包/辅领塞渗透率 |
| 效率 | Gross美补率、商补撬动系数（商补/美补） |
| 用户 | 持券渗透率、人均持券张数、核销率 |

### 5. 页面/频道 — "在哪儿转化"
| 模块 | 核心指标 |
|-----|---------|
| 闪购金刚 | 交易用户数、DAU、CTR、CXR |
| 猜你喜欢 | 交易用户数、曝光PV占比、CTR、CXR |
| 外卖feeds | 交易用户数、流量曝光占比、CTR、CXR |
| N3频道 | DAU、00后占比、领券人数、领券用户CXR |

### 6. 通用补贴/履约 — "经营健康度"
| 指标类型 | 核心指标 |
|---------|---------|
| 补贴效率 | 补贴订单单均美补金额、运营组补贴订单单均补贴金额 |
| 品牌合作 | 标品品牌补贴率、免配订单占比 |
| 履约体验 | 订单推单完成率、平均履约时长 |

## 二、三条战略主线

| 主线 | 核心问题 | 关联模块 |
|-----|---------|---------|
| **用户年轻化** | 如何提升90后/00后占比？ | 用户增长 + N3频道 + 域外流量 |
| **流量自给率** | 多少增长来自"内生能力"vs"外部输血"？ | 流量运营（剔除外投）+ 域外流量 |
| **补贴ROI** | 每花1块钱补贴，能带来多少GMV和留存？ | 权益运营 + 用户增长（CAC） |

## 三、品类→指标模块映射规则

执行 WBR 时，根据品类类型确定核心关注指标：

| 品类类型 | 核心关注模块 | P0指标 | P1指标 |
|---------|------------|--------|--------|
| 酒水 | 用户+权益+治理 | GTV、订单量 | 新客、风险水位、补贴ROI |
| 生鲜 | 流量+用户+履约 | GTV、订单量 | 新客、履约时长、损耗率 |
| 手机/3C | 流量+用户+供给 | GTV、订单量、客单价 | 新客、库存覆盖、价格力 |
| 小家电 | 流量+用户+场域 | GTV、订单量 | 新客、CXR、频道渗透 |
| 日百 | 用户+权益+供给 | GTV、订单量 | 复购率、补贴ROI、门店覆盖 |
| 通用/未知 | 全模块扫描 | GTV、订单量 | 取WoW变化最大Top指标 |

## 四、指标异动与品类动作的关联逻辑

品类WBR不只看品类自身指标，还要理解品类动作对上层指标的传导：

```
品类动作（如投券、选品）
    ↓ 直接影响
品类GTV/订单量/新客
    ↓ 向上传导
闪购整体GTV/DAU/CXR
    ↓ 交叉影响
其他模块指标（如补贴率、流量效率）
```

**跨模块关联检查**（输出时使用）：
- 品类补贴动作 → 可能影响 权益模块的Gross美补率
- 品类新客获取 → 可能影响 用户增长模块的CAC和结构
- 品类供给变化 → 可能影响 履约模块的完成率和时长

# 分析模块方法论库（CEO视角版）

本文件是 CEO Review Advisor 的提问弹药库。
根据材料场景激活对应模块，利用其中的框架和追问方向生成高质量追问。

> **使用原则**：方法论是底层判断逻辑，追问句式要用CEO式口语输出——简短、带判断、结合具体数字。

---

## 模块 X：纵向一致性检验

**触发场景**：年度运营计划 / 战略规划 / 任何包含目标→策略→举措→指标多层级结构的材料（复杂材料必做）

### 追问维度

1. **战略目标→业务策略**：策略是否承接战略目标？（Why→What 断点）
2. **业务策略→具体举措**：举措是否匹配策略方向？有没有"说一套做一套"？（What→How 断点）
3. **举措→指标可测量性**：每个核心举措是否有对应的可追踪输入指标？
4. **输入指标→输出指标**：因果链是否成立？输入指标改善能否合理推导出输出指标提升？
5. **资源分配与策略对齐**：钱和人是否流向核心策略？资源分配是否与优先级表述一致？

### CEO式追问

1. 从战略目标到具体举措，哪一层的承接最弱？断点在哪里？
2. 核心举措的输入指标是什么？团队是否真正在管理这些输入？
3. 「策略优先级和资源分配放在一起对比，是否一致？说是重点投入X，但预算里X占多少？」
4. 「每个里程碑有没有对应的可量化目标，还是只有动作描述？」

## 模块 S：市场与竞争审查

**触发场景**：战略方向评审 / 竞争分析 / 行业判断

### 追问维度

1. **战略逻辑**：Why this direction——为什么是这个方向，而不是其他？
2. **市场时机**：Why now——为什么是现在？早了还是晚了？
3. **竞争态势（BML）**：在关键能力维度上，我们 Beat / Meet / Lose 竞对的分别是什么？
4. **外部因素 vs 内生因素**：增长有多少来自行业顺风，多少是真正的自身能力？
5. **破坏性威胁**：是否有低端/新市场颠覆者正在积累势能，尚未被重视？

### 核心框架

- **三层四面**：行业大盘 / 线上大盘 / 美团 × 客/量/额/利
  > 先判断增长来自哪一层，再判断是结构性还是周期性
- **BML 竞争能力对标**：针对核心竞争维度（产品体验、供给密度、价格力、履约效率、品牌心智等），逐项判断与主要竞对的相对位置：
  - **Beat**：我们明显领先，形成差异化壁垒
  - **Meet**：与竞对持平，无显著优劣势
  - **Lose**：我们处于劣势，存在被替代风险
- **外部因素 vs 内生因素**：业务表现多少来自行业顺风（外部），多少是自身能力（内生）？
- **护城河体系**：网络效应 / 成本优势 / 转换成本 / 无形资产 / 高效规模
- **破坏性创新**：是否有低端 / 新市场颠覆者正在积累势能？

### CEO式追问

1. 「竞争角度看，CQT、MT、D+分别是多少绝对值和份额？增量市场里我们拿了多少？」
2. 「增速这么猛，多少是行业顺风，多少是自己做对了？」
3. 「在用户最在乎的维度上，我们是Beat还是Lose？Lose的维度过去一年趋势如何？」
4. 「护城河过去12个月是变宽了还是变窄了？证据是什么？」
5. 「谁是最危险的颠覆者？他们现在在哪个阶段？」

> **CEO补充外部信息的习惯**：在追问竞对时，常直接贴出自己掌握的外部信息，然后问「我们怎么看？」
> 例：「XHS这边可能有低估，2024年Q1-3的医美和医疗广告消耗口径是5.5、3.3……所以可能全年看比我们这边估计的要高40%左右，增速快。但是好在看起来主要是抢的百度的份额？」

## 模块 B：资源配置与预算

**触发场景**：年度预算 / 投资决策 / 资源分配

### 追问维度

1. **机会成本**：这笔资源最好的替代用途是什么？
2. **ROI 合理性**：回报是否覆盖资本成本？与其他投向相比如何？
3. **杠杆效应**：平台型投入 vs 一次性消耗？能否形成复利？
4. **风险结构**：是否判断过短期、中期、长期的风险，我们现在的对风险的判断是否过于乐观？
5. **J 曲线判断**：新业务/新投资处于亏损期是否合理？拐点的前置信号是什么？
6. **资源完整性**：从钱的角度看，有多少资源还没被统筹进来？是否存在隐性投入没被算清楚？
7. **跨业务联动**：这笔资源的投入，和其他业务线/平台有没有协同或冲突？

### 核心框架

- **机会成本 + 边际分析**：每一分资源，最好的替代用途是什么？边际投入的边际回报是否递减？
- **ROI 框架**：投入资本回报率，重点比较 ROI vs WACC
- **实物期权思维**：不确定性高时，分阶段投入 > 一次性押注；保留退出权的价值
- **J 曲线判断**：新业务/新投资是否处于正常亏损期？亏损拐点的前置信号是什么？


### CEO式追问

1. 「这个方向看起来是我们25年投入的重要方向，需要进一步阐述清楚——所需的资源投入以及节奏规划是什么？」
2. 「我们目前有什么案例验证过可行性？投入了多少补贴资源和流量曝光，做到了什么程度？」
3. 「这个亏损的改善完全线性，看起来主要是由于补贴率这一项的变化带来的——补贴率的变化是因为平销增长带来的，还是什么带来的？」
4. 「相同资源投入其他地方，ROI是否更高？」
5. 「从整个资源分配看，还有多少预算没被统筹进来？」

### 判断式追问示例（≤2条，先亮观点再施压）

- 「这个钱挺难赚的——做这个业务的核心逻辑是什么，为什么现在值得投？」
- 「投入产研多大规模，投入力度和预期ROI说清楚了吗？」

## 模块 N：新项目评审

**触发场景**：新业务立项 / 新产品投入 / 新市场进入

### 追问维度

1. **Why Now**：时机判断——为什么是现在做，而不是更早或更晚？
2. **陈三角**：机会 / 资源 / 组织能力，最薄弱的那一角是什么？补强路径是什么？
3. **PMF 信号**：用户需求真实性验证——是数据说明需求存在，还是团队在"假设"需求？
4. **市场规模**：TAM/SAM/SOM 与三层四面交叉验证，警惕 TAM 膨胀
5. **期望值与底线**：最坏情景下公司能承受吗？容忍底线是什么？
6. **VNOS**：用户安装和使用的理由是什么？是否足够犀利到发生迁移？（产品价值 = 新体验 - 旧体验 - 迁移成本）
7. **时间节点校验**：在哪个时间点、哪个指标应该出现拐点来验证判断成立？如果 T+N 数据不达预期，触发机制和退出条件是什么？
8. **加盟/自营扩张的选择和关注点**
8a. **加盟/渠道扩张能力链**（适用于依赖加盟商或渠道伙伴扩张的新业务）：招募能力（客资池从哪来？加盟商转化漏斗是什么？）/ 督导管理能力（人均管店数是否可承接？）/ 供应链响应能力（补货时效、城市覆盖、最小起订量）/ 淡旺季现金流（旺季数据不代表模型成立，加盟商淡季能否存活？）——加盟体系中任何一环的断点都会导致规模化失败。
8b. **自营规模化复制能力**（适用于直营/自营扩张模式）：
   - 自营前提判断（必须先问）：作为大公司，我们的规模优势是否真正转化为自营的成本优势，还是用规模掩盖了低效？作为平台，深入自营会改变供给侧合作意愿和产业链格局——这个代价被算清楚了吗？自营的边界在哪里，什么情况下平台模式比自营更优？
   - 标准化可复制性：试点依赖哪些不可复制的条件（点位稀缺性、关键人、本地关系）？去掉这些条件，模型还成立吗？
   - 成本曲线拐点：规模效应在第几家店/单元出现？固定成本摊薄和变动成本控制的节点分别是什么？
   - 人才复制速度：扩张所需的管理人才（店长/区域经理）培养周期多长？人才供给是否是扩张瓶颈？
   - 淡旺季自担风险：自营模式下淡季亏损由公司承担，淡季现金流压力对扩张节奏的约束是什么？
9. **试点验证焦点**：试点阶段是否在同时验证多个相互干扰的假设？（如：鲜啤 + 散白酒同时测，无法判断哪个是驱动因子）试点的核心问题是什么——是验证UE、品类选择、选址模型，还是供应链？焦点不清晰的试点，结论不可用于指导规模化。

### 核心框架

- **陈三角（美团内部）**：机会 / 资源 / 组织能力，三角缺一角则不立项
- **PMF 判断**：40% 失望测试 + Retention 曲线拐点，区分"可以做"和"应该做"
- **TAM/SAM/SOM**：市场规模分层，警惕 TAM 膨胀（与三层四面交叉验证）
- **Jobs-to-be-done**：用户真正要"雇佣"这个产品/服务完成什么任务？
- **期望值框架**：EV = Σ(概率 × 回报)，重点测试最坏情景的容忍底线
- **VNOS 方法论**：产品价值 = 新体验（New experience）- 旧体验（Old experience）- 迁移成本（Switching cost）。用于量化一个产品或新功能对用户的价值——只有价值大于零且足够显著，用户才会发生迁移。
- **专注第一块敲门砖（CEO）**：一个好的App，不是因为它应用多，而是因为它是一个很强的应用。先把第一块敲门砖打好，雪球才会滚动起来。

### CEO式追问

1. 「PMF信号出现了吗？还是我们在假设用户需要它？」
2. 「陈三角：机会/资源/组织能力，最薄弱的那一角是什么？补强路径是什么？」
3. 「用户迁移的第一理由是什么？是否足够犀利」
4. 「这个市场是我们进入才能做大，还是本来就存在等我们进入？」
5. 「最坏情景下公司能承受吗？容忍底线是什么？」
6. （加盟扩张场景）「竞对旺季日销是X，我们目标比它高20%——差距来自什么具体能力？不是规划，是已验证的能力。」
7. （加盟扩张场景）「旺季数据漂亮，但淡季日销跌幅是多少？加盟商淡季现金流能撑住吗？这才是加盟模型是否成立的真正检验。」
8. （加盟扩张场景）「第一批加盟商从哪来？目前蓄水了多少客资？加盟商招募能力是扩张能不能跑起来的前提，现在的漏斗在哪？」
9. （自营扩张场景）「我们做自营的成本优势是真实的，还是用规模补贴掩盖了低效？剔除平台流量和规模补贴之后，自营单元的真实成本结构是什么？」
10. （自营扩张场景）「平台入场自营，供给侧合作方会怎么反应？产业链格局的变化被算进决策里了吗？自营的边界在哪——什么情况下该退回平台模式？」
11. （试点验证场景）「试点阶段同时在测鲜啤和散白酒，那这个试点到底在验证什么？两个品类受众、供应链、选址逻辑完全不同，同时测的结果只能告诉你'合在一起行不行'，而不是'哪个方向该压注'。」

## 模块 M：指标设计

**触发场景**：KPI/OKR 制定 / 考核体系设计 / 数据看板搭建

### 追问维度

1. **指标完整性**：输入/输出指标是否配对？是否有约束指标？
2. **可控性**：输入指标是否真正在团队可干预范围内？
3. **代理问题**：指标被优化到极致，业务会变好还是变形？
4. **层层拆解**：价值树是否覆盖所有核心驱动因子？有没有遗漏的分支？
5. **领先性**：指标能否提前预警，还是只能事后确认？
6. **口径定义**：数字的统计口径是什么？有没有把不同性质的东西混在一起计算？
7. **主R/主S清晰度**：这个业务到底是主R（Revenue）还是主S（Scale）？目标设置是否与此一致？

### 核心框架

- **输入 / 输出指标体系**：
  - 输出指标（结果）：GMV / 利润 / NPS，反映结果但滞后
  - 输入指标（可控）：供给密度 / 响应时长 / 内容更新率，可干预且领先
  - 核心原则：**管理输入，观测输出**
- **价值树（McKinsey）**：从顶层目标逐层拆解驱动因子，确保指标覆盖完整
- **委托代理（Jensen）**：指标是否制造代理问题？（如：只看 GMV 导致低质订单）

### CEO式追问

1. 「咱们到底是主R还是主S？这个目标的设置可能会让业务有点困惑」
2. 「直接以DTU和GTV导向会不会忽略了用户维度？留存复购是否应该严格作为一个配对指标？」
3. 「这里的QV是仅首页综搜，不包含频道搜和商家页内搜等等这些？」（口径追问）
4. 「这三个分类不MECE——漏掉了X，所以这些100%的柱子是怎么理解？」
5. 「各里程碑和定量目标有没有关联起来，还是只有动作没有数字？」

### 判断式追问示例（≤2条，先亮观点再施压）

- 「DAU不一定是第一指标——这个业务真正要管的结果是什么？」
- 「会员成长值只看消费金额，但商业价值应该看毛利/净利，现在的设计有没有这个意识？」

## 模块 R：复盘归因

**触发场景**：季度/年度复盘 / 经营周报 / 未达目标分析 / 超额完成分析 / **经营现状多维分析**

> **重要：先判断材料子类型，再选追问模式**
>
> - **归因型**（有明确好坏、要解释为什么）→ 使用下方"归因追问维度"
> - **现状描述型**（多维拆解数据、呈现经营快照，不以论证某个命题为目的）→ 使用下方"现状型追问框架"

---

### 现状型追问框架（适用于经营现状/多维拆解类材料）

CEO 看现状型材料，核心问题不是"你的论证有没有漏洞"，而是**"你让我看清了什么、我应该关注什么、材料缺了哪些关键视角"**。

**四个追问方向：**

1. **结构完整性**
   - 关键指标是否给出了对大盘的贡献比例（ATU占比、订单量占比、GTV占比、收入占比等）？没有这个，管理者无法感知"这个群体有多重要"
   - 是否有核心品类/维度缺失？材料里已显现重要的品类是否每个都单独细拆？
   - 整体判断（总结锚点）是否缺失？是否在数据铺陈后有明确的战略指向

2. **分解充分性**
   - 趋势图是否只给了YoY，没有绝对值？绝对值小的波动用百分比看会严重失真
   - 品类/维度/时间线是否拆到位？上下半年差异大有没有给解释？
   - 增长来源是否拆清楚了？是自然增长还是补贴驱动，没拆就无法判断可持续性

3. **口径一致性**
   - 相似品类/指标是否存在口径重叠或错拆？（如：闪购鲜花店 vs 综合鲜花绿植店是否应合并？）
   - 同一指标在不同维度下口径是否统一？

4. **结论与行动锚点**
   - 现状描述之后，是否给出了明确的经营判断（增长品类怎么维持、下跌品类怎么诊断、非强竞争业务定价策略等）？
   - 业务机会判断是否有总体锚点在先，再分条展开？没有总锚点会让建议显得碎

**现状型CEO式追问示例：**
- 「这个群体的ATU占比、GTV占比、LTV1占比各是多少？开头没有这个，后面读下来没有量化的重要性感知」
- 「到餐品类渗透率这么高，为什么没有单独细拆？」
- 「趋势图只放了YoY变化，没有绝对值——如果基数小，YoY的大幅波动实际意义可能有限，应该怎么看？」
- 「休闲娱乐增速好看，但没拆清楚是补贴驱动还是自然增长，这个增速是否可持续无法判断」
- 「分月LTV放的是分城市级别的拆解，意义不大——城市级别差异反映的是消费能力，不是业务止血进度。应该分业务看分月LTV收敛曲线」
- 「业务机会判断直接列条目，缺少总的判断锚点——2026年大学生群体的核心命题是LTV改善还是规模扩张，先把这个说清楚」

### 归因型追问维度

1. **数字可信度**：口径是否一致？是否存在异常值或比较基准偏差？
2. **外部因素 vs 内生因素归因**：增长/下滑中，外部顺风占多少，内部执行占多少？
3. **输入指标下钻**：结果变差了，哪个输入指标是根因？是否可干预？按品类/城市/渠道等维度拆过吗？
4. **周期判断**：是结构性变化还是周期性波动？判断依据是什么？
5. **行动导向**：归因结论能否直接指导下一步行动？无法指导行动的归因等于无效
6. **双环反思**：不只问"执行对不对"，还要问"目标本身是否需要修正"
7. **暗点缺失**：材料是否只讲了亮点？暗点是什么？针对暗点有什么改善举措？
8. **时间节点校验**：上一期复盘里定下的改善举措，在这一期的时间节点上是否验证有效？如果未达预期，归因是否区分了"举措失效"和"时间未到"？下一个验证节点是什么时候、看什么指标？

### 核心框架

- **外部因素 vs 内生因素归因**：增长/下滑有多少是行业顺风/逆风（外部），多少是自身执行（内生）？
- **三层四面拆解**：从行业→线上→美团逐层定位偏差来源
- **5 Whys 根因分析**：连续追问 5 层，找到可干预的根因
- **反事实检验**：如果什么都不做，结果会是什么？用于剥离外部因素
- **双环反思**：不只复盘"做对了什么"，还要复盘"目标本身是否正确"

### CEO式追问

1. 「复盘里更多highlight的是亮点，没看到暗点——暗点是什么？尤其是去年最大的暗点，今年有什么改善或扭转的举措？」
2. 「增速这么猛，多少是行业顺风，多少是自己做对了？」
3. 「重点不是yoy高低，而是yoy和渗透率的关系——这两个一起拆过吗？」
4. 「归因结论会导致我们下一步做什么不同？无法指导行动的归因等于无效」
5. 「如果分商品词类型看，哪类表现低于大盘？分品类（正轻快）、分场域的角度拆过吗？」

### 判断式追问示例（≤2条，先亮观点再施压）

- 「重点不是yoy高低，而是yoy和渗透率的关系，拆过吗？」
- 「增速这么猛，多少是流感/节假日这类外部因素，多少是自己做对了？」

## 模块 O：组织与执行

**触发场景**：战略落地 / 组织调整 / 执行障碍诊断

### 追问维度

1. **权责对等**：有责无权必失败；有权无责必腐败——现在是哪种情况？
2. **执行障碍根因**：障碍来自能力不足、意愿不足，还是系统设计问题？
3. **战略对齐**：战略选择瀑布五层，哪一层是当前最薄弱的环节？
4. **战时/平时匹配**：当前处于战时（生存威胁）还是平时（增长扩张）？管理风格是否匹配？
5. **Plan B**：主路径失败的触发条件是什么？备选方案是否存在且已明确？
6. **人才承接**：这个方向有没有合适的人来承接？人才是决定成败的关键。

### 核心框架

- **战略选择瀑布（Roger Martin）**：赢的渴望 → 战场选择 → 制胜方式 → 能力要求 → 管理系统，五层对齐
- **战时 / 平时切换（Horowitz）**：战时（生存威胁）vs 平时（增长扩张）需要不同管理风格
- **权责对等**：有责无权必失败；有权无责必腐败
- **CEO组织能力观**：打造一个既有理性主义，又有真情实感且战无不胜的团队；绝缘体/半导体/导体——要做好区分

### CEO式追问

1. 「这个方向有没有合适的人来承接？人才是决定成败的关键，在资源充足的前提下」
2. 「战略选择瀑布五层是否对齐？哪一层是当前最薄弱的环节？」
3. 「现在是战时还是平时？当前的管理风格匹配吗？」
4. 「执行障碍的根源，是能力问题、意愿问题，还是系统设计问题？」
5. 「主路径失败的触发条件是什么？Plan B是什么？」

# 商业租赁合同专项上下文

适用范围：办公室租赁、厂房租赁、商铺租赁、仓库租赁、联合办公协议。（住宅租赁参考适用，但注意住宅受《民法典》第734条优先续租权等额外保护。）

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **租赁标的描述** | 地址、楼层、面积（建筑面积/使用面积/计租面积）、附属设施 |
| **租期与起租日** | 租期起止日、免租期（装修期）性质与长度 |
| **租金结构** | 基础租金、物业费、停车费、租金递增机制（年递增率）|
| **押金条款** | 押金金额（月数）、扣划条件、退还时限与方式 |
| **装修权利** | 装修审批要求、装修标准限制、装修押金、恢复原状义务 |
| **转租与分租** | 转租是否需房东书面同意、关联方转让是否豁免 |
| **提前退租** | 提前退租通知期、违约金计算方式（剩余租金 vs. 固定金额）|
| **续租权利** | 优先续租权、续租谈判期限、续租价格确定方式 |
| **房屋维修责任** | 结构性维修（房东）vs. 日常维护（租客）的边界 |
| **经营限制** | 行业限制、竞争性业态限制（商场多见）|
| **不可抗力与政策调整** | 因政府政策（如封控、征收）导致无法使用的处理 |

## 二、风险评估专项检查清单

### 2-A 租客（承租方）立场

#### 🔴 自动触发高危

1. **面积计量陷阱** ⚡
   - 合同写"建筑面积"但报价对标"使用面积"——差距通常 20-30%（公摊面积）
   - 未明确计租面积的测量方法和依据，房东可单方调整
   - 建议：合同明确约定"以实测使用面积为准"并附平面图

2. **免租期性质不明** ⚡
   - 免租期仅为"优惠"而非合同约定，房东提前终止后不退还
   - 免租期与租期的关系不明（免租期是否计入租期总时长）
   - 若提前退租，是否须"偿还"免租期对应租金

3. **提前退租违约金过重** ⚡
   - "剩余租期全额租金"作为违约金（实质是对租客的无限惩罚）
   - 正常范围：3-6 个月租金；超过 6 个月须重点标注
   - 房东无义务减损（不重新出租情况下仍要求全额赔偿）

4. **押金退还条件不明或期限过长** ⚡
   - 扣划条件由房东单方认定（如"房屋有任何损耗"）
   - 退还期限超过 60 天或以"完成验收"为前提（验收可被无限拖延）
   - 无利息约定（押金通常相当于 2-3 个月租金，资金成本不低）

5. **恢复原状义务无上限**
   - 租客须将房屋恢复至"原始状态"，但入驻时未做书面现状确认
   - 合同终止时可能面临超出装修成本的"恢复"要求

6. **转租/关联方使用被完全禁止**
   - 集团内部调整（如子公司使用）也触发转租条款，导致违约
   - 建议：明确豁免"向租客全资子公司或控股关联方的使用转让"

#### 🟡 中等风险（租客视角）

- 租金递增条款过高（超过 5%/年须关注）
- 物业费由房东单方调整，无上限约定
- 续租优先权须在到期前 X 个月行使，期限过短（少于 3 个月）
- 房屋维修责任不清晰，日常损耗由租客承担

---

### 2-B 房东（出租方）立场

#### 🔴 自动触发高危

1. **租金支付保障不足** ⚡
   - 无连续欠租的解除权（建议：连续欠租超过 15 天即可解除）
   - 押金不足以覆盖违约成本（押金低于 2 个月租金时风险较高）

2. **提前退租违约金过低**
   - 仅需提前 1 个月通知即可退租，且无违约金
   - 优质租客流失后重新招租的空置成本未被覆盖

3. **转租约束不足**
   - 租客可将房屋转租给任意第三方，房东无法控制实际使用人
   - 转租后实际使用人的违规行为（污染、超载等）责任归属不清

#### 🟡 中等风险（房东视角）

- 恢复原状义务约定模糊，租期结束后难以主张
- 装修审批要求流于形式，实际改造影响房屋价值
- 续租优先权给予租客的谈判期过长，影响房东重新招租计划

## 三、合规检查专项清单

### 《民法典》租赁合同强制性规定

- [ ] 租赁期限不超过 20 年（超出部分无效，《民法典》第705条）
- [ ] 不定期租赁：双方均可随时解除，但须给予合理通知期（第730条）
- [ ] 出租人出售房屋须提前通知承租人，承租人有同等条件优先购买权（第726条）
- [ ] 房屋抵押后新签租赁：买受人取得所有权后合同继续有效（"买卖不破租赁"，第725条）

### 商业用途合规

- [ ] 租赁用途与房屋规划用途一致（住宅不得用于经营，商业用途须为商业用地）
- [ ] 二手房转租：确认出租方有权转租（一手租约中允许转租，或取得房东书面同意）
- [ ] 经营性租赁：工商登记地址须与租赁地址一致

## 四、关键谈判要点

**租客应争取的条款：**
- 免租期以书面形式写入合同（而非口头承诺）
- 押金退还：合同到期/终止后 **30 天内**退还，无争议部分无条件退还
- 提前退租违约金上限：不超过 **3 个月**租金
- 关联方使用豁免条款
- 入驻前**书面确认房屋现状**（照片存档）

**房东应争取的条款：**
- 连续欠租 **15 天**触发解除权
- 装修须提交方案并书面审批，完工后验收
- 转租须书面同意，转租收益超出原租金部分按比例分享

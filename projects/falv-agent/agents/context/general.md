# 通用合同上下文（兜底）

适用范围：无专项 context 文件的合同类型，或用户明确选择"通用"的情形。

---

## 一、通用高风险触发清单

在无专项清单的情况下，风险评估师重点检查以下通用高危情形：

1. **合同标的不明确**：货物/服务描述模糊，无规格/质量/范围界定
2. **价款或计价方式不明确**：总价或单价不固定，无确定计算方式
3. **验收标准缺失或由一方单方决定**
4. **付款条件不明确**：无触发条件、无时间节点
5. **争议解决条款缺失或无效**（仲裁条款不规范；约定无管辖权的法院）
6. **违约金畸高或畸低**（《民法典》第五百八十五条）
7. **单方解除权无限制**
8. **知识产权归属不明**
9. **不可抗力定义极端（过窄或过宽）**
10. **无限连带责任**（《民法典》第六百八十六条）
11. **永久保密条款**（无合理期限）
12. **合同期限或续约条款不明确**

---

## 二、立场说明（通用）

| 立场 | 重点关注 |
|---|---|
| **甲方（委托/发起方）** | 付款触发条件是否可控、对乙方履约的约束力、知识产权归属、违约救济充分性 |
| **乙方（执行/接受方）** | 付款保障与账期、验收标准是否公平、责任上限是否合理、知识产权归属、退出机制 |
| **平衡** | 均等审查，标注利弊各方 |

---

## 三、通用合规必查项

- [ ] 双方主体资格合法（营业执照经营范围覆盖合同标的）
- [ ] 合同标的合法（不违反法律、行政法规的强制性规定）
- [ ] 签署方式有效（法定代表人签字 + 公章，或有授权委托书）
- [ ] 争议解决条款合法（仲裁机构存在；约定诉讼时管辖法院有管辖权）
- [ ] 无明显格式条款无效情形（《民法典》第四百九十七条）

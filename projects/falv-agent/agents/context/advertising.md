# 广告协议专项上下文

适用范围：广告投放协议、媒体广告合同、广告代理协议、信息流广告服务协议、KOL/MCN 合作协议、品牌植入协议、效果广告协议（CPC/CPM/CPA）。

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **广告资源与排期** | 广告位置、规格、投放时段、排期确认机制 |
| **计费方式** | CPM（千次展示）/ CPC（点击）/ CPA（转化）/ 包天/包周定价 |
| **效果承诺** | 展示量保量、点击率保底、转化目标承诺 |
| **数据报告** | 曝光数据、点击数据、转化追踪、数据核对机制 |
| **广告审核** | 广告内容审核标准、审核时限、拒绝投放的处理 |
| **违规内容责任** | 广告违法的责任分担（广告主 vs 媒体/平台）|
| **排他条款** | 竞品广告排他期、同行业广告限制 |
| **退款与补量** | 未完成量的补偿方式（补量 vs 退款）|
| **知识产权授权** | 广告素材授权范围、使用期限、二次传播权 |
| **结算方式** | 预付/后付、对账周期、发票要求 |

## 二、风险评估专项检查清单

### 2-A 广告主立场

#### 🔴 自动触发高危

1. **效果数据无法独立核验** ⚡
   - 数据完全依赖媒体方自有系统，无第三方监测机制（如秒针、AdMaster）
   - 合同不允许广告主接入第三方检测代码
   - 数据异常时的争议解决机制缺失（应约定第三方检测数据为准）

2. **"未完成量补量"而非退款** ⚡
   - 未投完的广告量只补量（延期投放），而非退款
   - 补量有效期过短（如 30 天），实际上无法用完
   - 未完成核心投放时段（如节日期间），补量无法恢复同等价值

3. **广告违规内容责任对广告主过重**
   - 未经广告主确认的素材变更导致违规，仍由广告主承担全部法律责任
   - 实际上媒体自动生成的广告内容（如 DSP 程序化广告），广告主难以事先审核

4. **无排他保证但收取了排他费用**
   - 约定排他时段内不投竞品，但无违约赔偿条款
   - 排他范围定义模糊（"同类产品"未界定），保护形同虚设

5. **合同主体与实际投放主体不一致**
   - 签约主体为代理商，实际媒体资源来自第三方，代理商无法保证履约
   - 广告主资金打入代理商账户后，代理商跑路风险无防范措施

#### 🟡 中等风险（广告主视角）

- 广告素材知识产权：媒体方有权对素材进行二次使用（如作为案例展示）
- 结算周期：后付账期过长（超过 60 天），占用广告主资金
- 暂停投放权：广告主发现效果不佳时能否随时暂停？费用如何结算？

---

### 2-B 媒体 / 代理商立场

#### 🔴 自动触发高危

1. **效果保底承诺过高，无法履行**
   - 承诺具体转化数量（销售量、注册量），而非仅承诺展示/点击
   - 无"不可抗力"豁免（如平台算法调整、行业竞争加剧）

2. **货款回收风险**
   - 广告主预算审批流程导致签约到付款周期过长（>90 天）
   - 无首付款要求，广告投完后广告主拒付

3. **广告内容审核责任**
   - 广告主提供违规素材（虚假宣传、无证经营），媒体方被连带处罚
   - 《广告法》第三十四条：广告经营者、发布者须查验相关证明文件
   - 合同中须明确：广告主保证素材合法合规并承担相应法律责任

#### 🟡 中等风险（媒体/代理商视角）

- 广告主随时撤稿权：已制作的创意/素材成本如何分担？
- 知识产权：媒体方基于广告主需求创作的素材归属（建议归广告主，媒体方保留案例展示权）

## 三、合规检查专项清单

### 《广告法》强制要求

- [ ] 广告内容不含虚假宣传（第四条、第二十八条）
- [ ] 特定行业广告须有资质（医疗、药品、保健品、金融需审批）（第三十八条至四十六条）
- [ ] 广告主须提供：营业执照、相关行政许可文件（广告经营者核验义务，第三十四条）
- [ ] 互联网广告须能够识别为广告（第二十六条）
- [ ] 不得利用未成年人作为广告代言人为特定商品代言（第三十八条第三款）

### KOL / 网红合作额外合规

- [ ] 博主/KOL 须在内容中标注"广告"或"商业合作"（《互联网广告管理办法》第九条）
- [ ] 广告主须对代言内容进行审核，不得指使代言人进行虚假代言
- [ ] MCN 机构的资质及连带责任约定

### 数据合规（程序化广告）

- [ ] 使用 Cookie / 设备 ID 进行精准定向的合法依据（PIPL 第十三条）
- [ ] 用户定向数据的使用限制（不得超出广告投放目的）

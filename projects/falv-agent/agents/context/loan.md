# 借款 / 民间借贷合同专项上下文

适用范围：企业间借款协议、股东借款合同、民间借贷合同（自然人之间）、商业贷款补充协议。（银行正式贷款合同通常为格式合同，审查重点不同。）

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **借款金额与币种** | 本金金额、币种、支付方式 |
| **利率条款** | 年利率/月利率、计息方式、复利约定 |
| **还款安排** | 还款期限、分期还款计划、提前还款权利 |
| **担保条款** | 担保类型（保证/抵押/质押）、担保范围、担保权实现方式 |
| **资金用途** | 借款用途限制、用途监督机制 |
| **违约事件** | 触发提前到期的情形、交叉违约条款 |
| **砍头息** | 预先扣除利息的约定（违法情形识别）|
| **罚息与复利** | 逾期罚息标准、复利（利滚利）约定 |
| **提前还款** | 提前还款权利、违约金或补偿金 |

## 二、风险评估专项检查清单

### 中国借贷法律红线（自动核查，任何违反均高危）

#### 🔴 利率红线（最高人民法院相关规定）

**民间借贷利率上限：**

| 利率范围 | 法律效果 |
|---|---|
| ≤ LPR × 4 倍（当前约 14%-15%）| 合法，法院支持 |
| > LPR × 4 倍 | 超出部分无效，法院不支持 |

> 注：LPR（贷款市场报价利率）每月公布，审查时须以**合同签署时**的 LPR 为基准计算。当前（2025年）一年期 LPR 约 3.5%，4 倍即约 14%。

**自动触发高危：**
- 年利率超过 LPR 4 倍 ⚡
- 月利率约定超过 LPR × 4 / 12 ⚡

#### 🔴 砍头息无效（《民法典》第670条）

- 预先从本金中扣除利息（借 100 万实际到账 90 万但须还 100 万本金）
- 常见形式：签署费、手续费、服务费等名目实质上是预扣利息
- 法律效果：应按实际到账金额认定借款本金

#### 🔴 复利约定（利滚利）限制

- 双方可约定将利息计入本金，但**复利不得超过 LPR 4 倍上限**
- 超出部分同样无效

---

### 2-A 借款人立场

#### 🔴 自动触发高危

1. **利率超过法定上限** ⚡（见上方红线）

2. **砍头息识别** ⚡
   - 合同金额与实际到账金额不一致
   - 各种名目的"费用"预先扣除

3. **担保条款过于宽泛** ⚡
   - 抵押/质押范围不清晰，可能覆盖核心资产
   - 保证人承担无限连带责任且无追索权约定
   - 交叉违约条款：其他债务违约也触发本笔借款提前到期

4. **提前还款被禁止或设置高额违约金**
   - 借款人无权提前还款，或须支付剩余利息总额作为违约金
   - 正常约定：提前还款须提前 30 天通知，支付 1-3 个月利息补偿

5. **资金用途限制过于模糊**
   - "不得用于违法活动"等空泛表述，出借人可随时以"违反用途"为由宣布提前到期

#### 🟡 中等风险（借款人视角）

- 逾期罚息约定超过正常标准（建议不超过合同利率的 1.5 倍）
- 担保物评估方式由出借人单方决定（影响担保物是否充足的判断）
- 违约事件定义过宽（如"财务状况恶化"等主观判断）

---

### 2-B 出借人立场

#### 🔴 自动触发高危

1. **无有效担保** ⚡
   - 仅凭信用借款，无抵押/质押/保证
   - 企业间借款：借款方资产状况不清晰，到期无力偿还风险高

2. **抵押/质押未完成登记** ⚡
   - 不动产抵押须在不动产登记机构办理登记（未登记不对抗第三人）
   - 股权质押须在市场监管局登记（《民法典》第443条）
   - 应收账款质押须在中国人民银行征信中心登记

3. **企业间借款合法性** ⚡
   - 非金融机构企业间借贷：《民法典》施行后原则上承认效力，但仍需注意：
     - 套取金融机构贷款再转贷：无效
     - 以"名为投资实为借贷"方式操作：司法认定存在不确定性
   - 民间借贷中，出借人须证明**款项已实际交付**（转账凭证是关键证据）

4. **还款来源不明确**
   - 合同未约定还款资金来源，借款人到期无力偿还时无追索路径

#### 🟡 中等风险（出借人视角）

- 利率约定接近上限，若 LPR 变化导致超限，超限部分无效
- 保证人资质未核实（个人保证人须确认其还款能力；企业保证人须确认未超越经营范围）
- 无借款用途监督机制，资金被挪用风险

## 三、合规检查专项清单

### 《民法典》借款合同强制规定

- [ ] 利率不超过 LPR × 4 倍（合同签署时）
- [ ] 预先扣除利息的，以实际到账金额认定本金
- [ ] 借款合同须为书面形式（自然人之间超过一定金额）

### 担保合规

- [ ] 不动产抵押：已在不动产登记中心办理抵押登记（《民法典》第402条）
- [ ] 动产质押：质押物已实际交付出质人（《民法典》第429条）
- [ ] 股权质押：已在市场监管局办理出质登记（《民法典》第443条）
- [ ] 应收账款质押：已在人民银行征信中心登记
- [ ] 保证合同：已明确保证方式（一般保证 vs. 连带保证）

### 税务合规（企业间借款）

- [ ] 利息收入须开具发票并缴纳增值税（税率 6%）
- [ ] 利息支出在企业所得税前扣除须满足独立交易原则（关联方借款须注意转让定价）
- [ ] 向境外关联方支付利息：须预扣所得税（10%）

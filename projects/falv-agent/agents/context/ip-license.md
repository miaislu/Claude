# 知识产权许可合同专项上下文

适用范围：商标许可协议、专利许可协议、版权（著作权）授权协议、软件许可协议（SaaS 以外的传统软件）、技术秘密许可、综合知识产权许可协议。

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **许可权利界定** | 许可类型（独占/排他/普通）、许可范围（地域/产品/用途）|
| **许可期限** | 许可有效期、到期续期条件 |
| **许可使用费** | 一次性付清 / 分期 / 销售额提成（Royalty）|
| **质量控制** | 被许可方使用标准、许可方监督权（商标许可必须有）|
| **转授权** | 是否允许再授权（sublicense）及条件 |
| **改良与衍生** | 被许可方改良成果的归属 |
| **无效/撤销风险** | 基础 IP 被无效/撤销后的处理 |
| **维权义务** | 第三方侵权时双方的维权责任与费用分担 |
| **保密义务** | 技术秘密的保密措施要求 |
| **备案与登记** | 商标许可须向商标局备案、专利许可可申请登记 |

---

## 二、许可类型核心区别（必须在分析中明确识别）

| 类型 | 含义 | 许可方能否再许可他人 | 被许可方能否独立起诉侵权 |
|---|---|---|---|
| **独占许可** | 全球/指定区域内仅此一家 | ❌ 不能 | ✅ 通常可以 |
| **排他许可** | 许可方自己可用，但不能再给第三方 | ❌ 不能给第三方 | ⚠️ 需合同明确 |
| **普通许可**（非独占）| 许可方可同时授权多家 | ✅ 可以 | ❌ 通常不能单独起诉 |

> ⚠️ 独占与排他的区别极易混淆——**合同用词不统一时须明确确认双方真实意图。**

---

## 三、风险评估专项检查清单

### 3-A 被许可方立场

#### 🔴 自动触发高危

1. **许可类型与对价不匹配** ⚡
   - 支付了独占许可价格，但合同写的是"排他许可"或"普通许可"
   - 许可方保留"在特定情形下向第三方再许可"的权利，实质削弱独占性

2. **基础 IP 权利状态不稳定** ⚡
   - 商标处于异议/撤销程序中（须核查商标局数据库）
   - 专利面临无效宣告请求（须核查专利复审委记录）
   - 许可方非 IP 权利人（须核查权属证明）
   - 基础 IP 被无效/撤销后，被许可方已支付费用的退还机制缺失

3. **改良成果归许可方** ⚡
   - 被许可方投入资源对许可技术进行改良，改良成果自动归许可方
   - 建议：被许可方独立改良成果归被许可方，共同改良按贡献比例共有

4. **质量控制条款对被许可方过于苛刻**
   - 许可方可随时以"质量不达标"为由终止许可，标准由许可方单方认定
   - 无明确的整改通知和整改期限（建议：书面通知 + 30 天整改期）

5. **商标许可未备案的法律风险**
   - 商标使用许可须在商标局备案（《商标法》第43条）
   - 未备案的许可合同对第三方不发生效力，被许可方在侵权诉讼中权利受限

#### 🟡 中等风险（被许可方视角）

- 许可地域范围不清晰（"中国境内"是否包括港澳台？）
- Royalty 计算基础不明确（净销售额 vs. 含税销售额 vs. 出厂价）
- 许可方维权不积极但又不允许被许可方自行维权，第三方侵权损害被许可方权益
- 被许可方无法获得 IP 权利瑕疵的赔偿（如专利被无效后已支付 Royalty 不退还）

---

### 3-B 许可方立场

#### 🔴 自动触发高危

1. **商标许可无质量控制条款** ⚡
   - 《商标法》第43条：许可方须监督被许可方的商品/服务质量
   - 无质量控制条款的商标许可可能被认定为"裸许可"，导致商标被撤销
   - **商标许可合同必须包含质量控制条款**

2. **Royalty 审计权缺失**
   - 按销售额提成但无审计权，被许可方可能低报销售额
   - 建议：许可方有权每年审计一次被许可方相关财务记录

3. **技术秘密泄露风险**
   - 许可的技术秘密无保密措施要求，被许可方对其员工和转包商无约束
   - 许可合同终止后技术秘密使用限制不明确

4. **独占许可导致自身无法使用**
   - 独占许可条款过宽（未保留许可方自身在特定场景的使用权）

#### 🟡 中等风险（许可方视角）

- Royalty 最低保底金额过低，不足以覆盖许可方的机会成本
- 被许可方改良成果许可方无法反授权使用（建议：双方互相授权使用对方改良成果）
- 再授权条件过于宽松，被许可方可将许可权转给竞争对手

## 四、合规检查专项清单

### 商标许可

- [ ] 许可合同须向商标局备案（《商标法》第43条第三款）
- [ ] 合同包含质量监督条款（否则面临商标被撤销风险）
- [ ] 被许可方须在商品/包装上标注"商标注册人授权使用"

### 专利许可

- [ ] 专利有效性核查（未过期、未被无效）
- [ ] 独占/排他许可可在国家知识产权局登记（有利于对抗第三方）
- [ ] 被许可方改良形成的改进专利归属已明确

### 跨境许可额外合规

- [ ] 向境外许可技术：须核查是否属于《技术出口目录》限制类（须商务部审批）
- [ ] 境外许可方向中国被许可方：Royalty 支付须代扣预提所得税（10%）
- [ ] 跨境许可须在商务部进行技术进出口合同登记

## 五、NDA / 保密协议中的 IP 条款专项

适用场景：独立保密协议（NDA）、合同内嵌保密条款、技术合作或尽调阶段保密安排。

### 5.1 技术秘密的定义边界

| 定义方式 | 风险评估 |
|---|---|
| 列举式（枚举具体类别） | 🟢 清晰，但可能遗漏新类别——建议加兜底"及其他标注保密的信息" |
| 泛化式（"所有交换信息均保密"） | 🔴 过宽，对披露方执行负担重，对接收方法律风险大 |
| 排除式（明确列出不保密情形） | 🟢 建议必须包含四类标准例外（见下方）|

**标准例外情形（缺少任一均构成起草缺陷）：**

1. 签署前已为接收方所知的信息
2. 已进入公众领域的信息（非因接收方违约）
3. 接收方从无保密义务的第三方独立获取
4. 接收方独立开发（须有证明负担规则）

### 5.2 保密义务的合理边界

**期限**：

- 永久保密条款 → 在中国法实践中效力存疑；建议约定固定期限（通常 2-5 年），商业秘密核心部分可约定"在构成商业秘密期间持续有效"
- 法条依据：《反不正当竞争法》第九条（商业秘密保护以构成商业秘密为前提）

**返还/销毁义务**：

保密协议终止后的处理须明确，否则接收方持续保留可能构成侵权风险：

```
建议文本（起草技术）：
"本协议终止后，接收方应于【X】日内，根据披露方书面选择，
将所有载有保密信息的材料予以返还或销毁，并出具书面确认。
接收方因归档目的留存的电子副本不受此限，但仍受本协议保密义务约束。"
```

### 5.3 NDA 中的 IP 归属陷阱

以下条款在 NDA 中极易被忽视，但可能构成重大 IP 归属风险：

1. **接收方基于保密信息作出的改良/衍生成果归属**
   - 问题：NDA 未约定，改良成果可能归接收方所有
   - 建议：明确"接收方在接触披露方保密信息后作出的、与该信息实质相关的改良成果，知识产权归属由双方另行协商；在未另行协商前，接收方不得实施或许可实施"

2. **NDA 过宽导致被认定为竞业禁止**
   - 问题：NDA 中禁止从事某类业务实质等同于竞业限制，须满足竞业禁止补偿要求
   - 建议：NDA 保密义务仅限于信息使用限制，不设置业务从事禁止

### 5.4 起草检查清单（NDA）

- [ ] 保密信息定义有四类标准例外
- [ ] 保密期限明确（非永久）
- [ ] 返还/销毁义务有触发机制和时限
- [ ] 接收方被允许向有知悉必要的雇员/顾问披露（须绑定保密义务）
- [ ] 接收方因法律要求被迫披露时有提前通知义务
- [ ] 接收方保密义务不等同于竞业禁止
- [ ] 泄露后的救济手段（申请禁令+损害赔偿）已约定
- [ ] 若涉及个人信息：须叠加满足 PIPL 合规要求

## 六、著作权转让专项

适用场景：软件源代码转让、美术作品转让、文字作品版权转让、音视频著作权转让、职务作品约定转让。

### 6.1 转让 vs 许可的本质区别

| 维度 | 著作权转让 | 著作权许可 |
|---|---|---|
| 权利归属 | 永久移转给受让方 | 著作权人保留，仅授权使用 |
| 控制权 | 受让方完全控制 | 著作权人可设置使用条件 |
| 再授权 | 受让方可自由再授权 | 须合同授权 |
| 对价通常 | 一次性买断 | 持续 Royalty 或分期付款 |
| 人身权 | **无法转让**（署名权、修改权、保护作品完整权） | 不涉及 |
| 适用法条 | 《著作权法》第二十五条（2021修正） | 《著作权法》第二十四条 |

⚠️ **关键：著作权人身权不可转让。** 转让合同不论措辞多宽泛，均无法转让作者的署名权、修改权和保护作品完整权。受让方需要的是：合同中明确约定作者不反对受让方修改作品（即在行使修改权时不提出异议）。

### 6.2 转让范围陷阱

**问题一：转让范围未穷举财产权子类型**

《著作权法》第十条列举了 16 项具体财产权利，转让合同应逐项确认转让范围，避免模糊的"全部著作权转让"措辞：

| 权利 | 是否常被遗漏 | 建议处理 |
|---|---|---|
| 复制权 | 否 | 标准转让 |
| 发行权 | 否 | 标准转让 |
| 信息网络传播权 | **经常** | 必须明确约定 |
| 改编权 | **经常** | 必须明确约定 |
| 汇编权 | 是 | 视需求约定 |
| 出租权 | 是 | 视需求约定 |
| 广播权 | 是 | 视需求约定 |

**问题二：转让条款未覆盖未来版本**

软件转让中常见陷阱：原始版本权利转让，后续迭代版本（衍生作品）权利归属未约定。

建议文本：
```
建议文本（起草技术）：
"受让方取得上述著作权财产权后，以上述原始作品为基础创作的
演绎作品（包括改编、翻译、注释、整理等）的著作权归属，由受让方
按本协议或另行协议约定独立处理，转让方对此无权主张。"
```

### 6.3 职务作品条款（雇主/雇员场景）

依据《著作权法》第十八条，职务作品分两类：

**普通职务作品**（大多数情形）：
- 著作权归作者（员工）
- 雇主有两年内优先使用权
- 合同另有约定的从约定

**特殊职务作品**（著作权归雇主）：
- 主要利用雇主物质技术条件创作的工程设计图、产品设计图、地图、示意图、计算机软件
- 法律、法规规定或合同约定的其他情形

**劳动合同/IP 归属协议建议条款：**

```
建议文本（起草技术）：
"在职期间，乙方利用甲方资源（工作时间、设备、资料）或与
甲方业务相关创作的一切作品（含计算机软件），著作权中的
财产权利归甲方所有；乙方的署名权依法保留，但同意不以行使
修改权或保护作品完整权为由阻止甲方对作品的合理商业使用。"
```

### 6.4 著作权登记

著作权自作品创作完成时自动产生，无需登记。但登记具有以下实践价值：

| 目的 | 说明 |
|---|---|
| 权属证明 | 著作权登记证书是争议中的初步权属证据 |
| 诉讼保全 | 向法院申请诉前禁令时，登记证书加强证明效力 |
| 对外授权 | 部分商业场景（如出版社、平台）要求提供登记证书 |
| 软件著作权 | 计算机软件著作权登记在中国信息技术安全测评中心办理 |

**转让合同建议约定**：受让方有权以自己名义申请著作权登记，转让方须配合提供必要文件。

### 6.5 著作权转让合规检查清单

- [ ] 转让权利逐项列举（含信息网络传播权、改编权）
- [ ] 明确作者署名权保留方式（如"作者署名权保留，但作者同意不阻止受让方的合理修改"）
- [ ] 约定职务作品归属（劳动合同或 IP 归属协议中）
- [ ] 约定后续演绎作品归属
- [ ] 约定受让方申请著作权登记时转让方的配合义务
- [ ] 约定转让对价及支付方式（若无对价，注意赠与合同的特殊规则）
- [ ] 若涉及软件：明确源代码交付义务和交付形式

# 采购合同专项上下文

适用范围：货物采购合同、原材料采购协议、设备购买合同、年度采购框架协议、战略供应商协议（含 IT 采购、MRO 采购）。

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **商品规格与标准** | 技术规格书、质量标准、样品确认程序 |
| **价格条款** | 单价、总价、价格调整机制（含原材料联动条款）|
| **最小起订量与预测** | MOQ、滚动预测、Binding/Non-binding Forecast |
| **交货条款** | Incoterms 条款、交货地点、交货时间、分批交货规则 |
| **验收标准** | 验收方法、不合格品处理（退换/返工/折价）、异议期限 |
| **付款条款** | 账期（Net 30/60/90）、付款方式、预付款/信用证 |
| **质量保证期** | 保修期限、缺陷责任期、质量追溯要求 |
| **库存与备货义务** | 供应商安全库存要求、缺货责任、超额库存处理 |
| **独家供应** | 独家供应条款、竞争性来源限制 |
| **知识产权** | 产品专利权属、侵权责任分担、定制产品归属 |
| **反腐合规** | 商业贿赂禁止、合规认证要求、审计权 |

## 二、风险评估专项检查清单

### 2-A 采购方（甲方）立场

#### 🔴 自动触发高危

1. **质量标准不明确，验收权利受限** ⚡
   - 质量标准仅参考"国家标准"而无具体技术规格书（国标是底线，不是质量基准）
   - 验收标准由供应商单方界定，或仅凭出厂检测报告为准
   - 异议期限过短（到货后 3 天内提出），实际上不够用于全面检验

2. **价格锁定机制缺失** ⚡
   - 年度框架协议未约定价格，每次下单需重新谈判
   - 价格调整条款由供应商单方触发（如"原材料价格上涨时可调整"），无明确触发标准和上限
   - 无竞争性报价权（供应商涨价后采购方无权拒绝且仍须采购）

3. **交货违约责任太轻** ⚡
   - 逾期交货仅需"尽快安排"，无逾期违约金
   - 供应商可单方取消订单且无赔偿
   - 紧急情况（节假日、年底）无保供承诺

4. **独家供应条款形成供应依赖**
   - 合同要求采购方只能向该供应商采购（无其他来源），供应商却无供货保证
   - 长期独家 + 无数量承诺 = 采购方既无保障又无替代

5. **知识产权风险未分配清楚**
   - 供应商提供的产品存在专利纠纷，合同未约定侵权赔偿由供应商承担
   - 定制开发的产品/工具归供应商所有，采购方只有使用权

#### 🟡 中等风险（采购方视角）

- 质量保证期过短（低于行业惯例）或起算点不合理（出厂日 vs 到货日）
- 不合格品退换周期过长，影响生产计划
- 付款条件中"开具发票后 XX 天"，供应商可拖延开票以实质加速付款
- 反腐合规条款缺失（大型企业合规要求）

---

### 2-B 供应商（乙方）立场

#### 🔴 自动触发高危

1. **账期过长且无欠款保障** ⚡
   - 付款账期超过 90 天（接近民法典规定的货款诉讼时效警戒线）
   - 付款以"甲方资金到位"为前提（资金未到位则不付款，将采购方内部问题转嫁供应商）
   - 无应收账款保障机制（如银行保函、应收账款保理可操作性）

2. **无限退货权** ⚡
   - 采购方可在验收期内以任何理由退货
   - 滞销品、超额库存可退回供应商（实质将采购方的销售风险转嫁）

3. **最小起订量（MOQ）义务不对等**
   - 供应商须保持安全库存（占用资金），但采购方无最小采购量保证
   - 预测（Forecast）全为 Non-binding，供应商备货后若不下单，损失全由供应商承担

4. **质量标准单方调整**
   - 采购方可单方提高质量要求而无需额外支付
   - 验收标准变更后仍按原价结算

5. **无限赔偿责任**
   - 因产品质量问题导致采购方下游损失（含召回成本、第三方索赔），供应商承担全额赔偿
   - 无责任上限条款（建议：上限为相关订单金额的 1-3 倍）

#### 🟡 中等风险（供应商视角）

- 交货时间要求过于严格（如 24 小时内），实际供应链无法保证
- 样品确认程序繁琐，批量生产前须多轮确认，但无对应的时间补偿
- 采购方审计权：可随时对供应商工厂审计，无预约要求，影响正常运营

## 三、合规检查专项清单

### 通用采购合同必查项

- [ ] 货物描述明确（型号、规格、数量、包装要求）
- [ ] Incoterms 条款明确（DDP/DAP/FOB等，明确风险转移时点）
- [ ] 质量标准可量化（非"符合国家标准"的模糊表述）
- [ ] 验收程序明确（地点、方法、期限、异议流程）
- [ ] 不可抗力条款合理（定义、通知义务、续履行 vs 解除选择权）

### 政府采购 / 国有企业采购（适用时）

- [ ] 采购程序合规（公开招标/邀请招标/竞争性谈判/单一来源采购的适用条件）
- [ ] 采购合同须在政府采购平台备案
- [ ] 合规条款：禁止商业贿赂、关联交易审批

### 跨境采购额外合规

- [ ] 进口关税归属（DDP 时由供应商负担；FOB 时由采购方负担）
- [ ] 原产地证明要求（享受优惠关税协定时）
- [ ] 进口许可证/配额要求（特定管制商品）
- [ ] 出口管制合规（供应商保证产品不含受限技术）

### 反腐合规条款（跨国公司/大型企业通常要求）

- [ ] 供应商声明未参与商业贿赂（《刑法》第一百六十四条）
- [ ] 供应商同意采购方审计权（合规审计）
- [ ] 禁止提供任何形式的回扣、礼品超过一定金额（通常规定不超过 300 元/次）

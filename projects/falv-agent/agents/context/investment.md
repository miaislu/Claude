# 投资协议专项上下文

适用范围：VC/PE 投资协议、Term Sheet、股东协议（SHA）、股权认购协议、可转债协议、股权转让协议（含并购）。

---

## 零、商业条款 vs 律师修改——投资协议分类参考

修改建议引擎在处理投资协议时，须按下表进行分类。**分类决定建议格式和读者**：商业条款需客户决策，律师修改可直接落地。

### 🏢 商业条款（需投资委员会/决策者拍板）

这类条款反映双方的利益博弈，没有客观正确答案，决策取决于投资方的风险偏好和谈判地位。

| 条款 | 典型商业决策点 |
|---|---|
| **回购利率** | 8%、10%、12%？单利还是复利？ |
| **优先清算权结构** | 参与型（Participating）vs 非参与型？有无封顶？ |
| **反稀释机制** | 完全棘轮 vs 广义加权平均 vs 窄基加权平均？ |
| **是否要优先股息** | 要不要固定股息率？多少？（如8%） |
| **合格 IPO 门槛** | 估值多少？融资额多少？接受哪些交易所？ |
| **一票否决权事项范围** | 哪些事项需要投资方同意？金额门槛是多少？ |
| **领售权触发机制** | 需双方同意还是可单方触发？触发条件是什么？ |
| **创始股东 Vesting 时间表** | 4年还是其他？有无cliff期？ |
| **限制交易期限** | 30个月还是更长？竞争对手名单如何界定？ |
| **新项目优先投资权** | 是否需要？投资比例如何定？ |

### ⚖️ 律师修改（可直接指示律师改，无需业务决策）

这类问题有客观正确的修改方向，属于起草技术问题。

| 条款 | 典型起草问题 |
|---|---|
| **Vesting 加速条款** | 标准条款缺失，有固定模板可补充 |
| **与公司对赌的执行程序** | 未说明须走减资程序，法律上执行有障碍 |
| **对赌指标措辞** | 目标已确定但表述不可量化/不可审计（写法问题，非目标本身） |
| **回购价款支付时限** | 40天/90天已有共识，但缺少逾期利息条款 |
| **优先购买权通知送达机制** | 通知期合理，但送达方式规定不完整 |
| **草稿空白项** | 关键数字/主体信息尚未填入 |
| **定义不一致** | 同一概念在不同条款中使用不同表述 |
| **跨条款引用错误** | 条款号引用有误 |
| **担保登记要求** | 质押/抵押条款未说明须办理登记方可对抗第三人 |
| **反稀释调整完成前的限制** | 已有共识但忘记写明"调整完成前不得发行新股" |

> ⚠️ **分类边界案例**：回购利率「12%」到底是商业还是技术？
> - 如果是「双方对利率是否接受12%尚未达成共识」→ 🏢 商业决策
> - 如果是「双方已同意12%但合同写成了年化12%复利（偏高）」→ ⚖️ 律师修改
> 判断依据：**分歧在于数字本身，还是数字的写法**。

## 一、专项条款类别（补充条款分析师的通用分类）

条款分析师在处理投资协议时，须在标准分类之外额外识别以下类别：

| 专项类别 | 典型条款内容 |
|---|---|
| **估值与融资结构** | Pre/Post-money 估值、融资金额、轮次定义、期权池调整 |
| **优先权条款** | 优先清算权（参与型/非参与型）、优先股息、转换权 |
| **反稀释保护** | 加权平均反稀释、完全棘轮反稀释、触发条件、豁免情形 |
| **控制权条款** | 董事席位、观察员席位、一票否决权事项清单 |
| **退出相关权利** | 领售权（Drag-along）、随售权（Tag-along）、优先购买权（ROFR）、共同出售权（Co-sale） |
| **锁定与归属** | 创始人锁定期、股份归属计划（Vesting）、离职条款（Good Leaver/Bad Leaver）|
| **回购与赎回** | 回购触发条件、回购价格计算、回购期限 |
| **对赌安排** | 业绩承诺指标、上市承诺、触发条件与后果、连带责任主体 |
| **信息权与监督权** | 财务报告频率、审计权、重大事项知情权 |
| **反稀释豁免** | 员工激励期权、战略融资豁免、已批准发行豁免 |
| **保密与排他** | 排他谈判期（Exclusivity）、保密范围和期限 |
| **交割先决条件** | 尽调完成、监管批准、章程修订 |

## 二、风险评估专项检查清单

### 2-A 投资方立场（重点保护投资人权益）

#### 🔴 自动触发高危的情形

1. **对赌指标不可量化或不可审计**
   - 使用"行业领先"、"快速增长"等模糊表述而非具体数字
   - 指标依赖内部数据且无独立审计机制
   - 考核基准年不明确

2. **回购条款可执行性存疑**
   - 回购价格计算依赖被投方提供的数据（无独立验证机制）
   - 回购主体仅为公司（公司资金不足时无法执行）；应同时约定创始人个人连带回购义务
   - 回购期限过长（超过 6 个月）

3. **优先清算权设计不合理**
   - 参与型优先权（Participating Preferred）无封顶（Cap），可能导致普通股持有人几乎无收益
   - 多轮投资时优先级排序不明确（先进先出 vs. 后进先出 vs. 按比例）

4. **反稀释保护缺失或过弱**
   - 无反稀释条款；或使用完全棘轮但豁免情形过宽
   - 加权平均公式未明确是"宽基"（Broad-based）还是"窄基"（Narrow-based）

5. **一票否决权事项清单不完整**
   - 缺少：处置核心资产、关联交易、修改章程、新增债务（超阈值）、股权激励超出期权池
   - 否决权可被章程修订绕过（应约定修改否决权事项需投资方书面同意）

6. **领售权触发条件过高或缺失**
   - 无领售权，导致投资方在创始人不配合时无法推动退出
   - 触发条件要求全体股东同意（等同于无效）

7. **创始人锁定期不足**
   - 锁定期短于 3 年，或 Bad Leaver 条款缺失（创始人不良离职后仍保留全部股权）

8. **信息权过窄**
   - 无季度/年度财务报告义务
   - 无重大事项（诉讼、监管处罚、核心人员离职）即时通知义务

#### 🟡 中等风险关注项（投资方视角）

- 对赌连带责任主体是否包含创始人个人（仅公司承担则风险较高）
- 优先购买权通知期限是否合理（过短则实际上无法行使）
- 可转债转换价格调整机制
- 跨境投资的外汇管制和审批要求
- VIE 架构合规性（如适用）

---

### 2-B 创始人（被投方）立场（重点保护经营自主权）

#### 🔴 自动触发高危的情形

1. **同意权清单过长且门槛过低**
   - 日常经营决策（如单笔超 X 万元合同）需投资方书面同意
   - 正常的人员招募/薪酬调整需书面同意
   - 实质上削弱经营团队对公司的控制权

2. **对赌指标脱离创始人控制**
   - 指标受宏观经济、行业周期或政策影响，非团队努力可决定
   - 若未能上市，单方赋予投资方回购权且无任何豁免机制（如市场系统性原因）

3. **反稀释机制不公平**
   - 完全棘轮反稀释（Full Ratchet）：下轮任何低价融资都将大幅摊薄创始人
   - 反稀释触发的豁免情形过窄（不豁免战略融资、收购、员工激励）

4. **Bad Leaver 条款过于严苛**
   - "Bad Leaver"定义宽泛，轻微过失也触发（应仅限重大违规/竞业）
   - Bad Leaver 股份按极低价格强制回购（如按历史成本）

5. **领售权触发门槛过低**
   - 持有少量股份的投资方即可触发领售权，强制创始人出售
   - 领售价格无底价保障，创始人可能被低价强制退出

6. **条款修改无须创始人同意**
   - 股东协议的修改仅需投资方多数同意即可，创始人无实质否决权

#### 🟡 中等风险关注项（创始人视角）

- 期权池（ESOP）设置时间点（Pre-money 设置会稀释创始人）
- 竞业限制期限和地域范围是否合理
- 融资顺序限制（是否需要投资方同意方可进行下轮融资）
- 知识产权归属（个人历史 IP 与公司 IP 的隔离）

## 三、合规检查专项清单（投资协议）

### 境内投资

- [ ] 投资方资质合法（基金登记、LP 资格等）
- [ ] 认购价格不低于净资产（国有企业参股时需注意）
- [ ] 股权结构变更的工商登记时限（完成交割后 30 日内）
- [ ] 股东协议与公司章程不冲突（章程具有对外公示效力，若冲突以章程为准）
- [ ] 对赌条款不违反《公司法》资本维持原则（与公司对赌的回购需符合法定减资程序）

### 外商投资（如适用）

- [ ] 商务部/市场监管外资备案或审批
- [ ] 外商投资负面清单核查
- [ ] 外汇登记（资本金账户开立、ODI/FDI 登记）
- [ ] VIE 架构的 VIE 协议合规性（如适用）

### 对赌条款效力（最高院裁判规则）

依据最高人民法院相关判决（以"海富投资案"为代表）的裁判规则：

| 对赌类型 | 效力 |
|---|---|
| 投资方 vs. 公司（公司承担补偿义务） | ⚠️ 可能损害公司和债权人利益，效力存疑；须通过合法减资程序执行 |
| 投资方 vs. 创始人个人（创始人个人承担） | ✅ 通常有效 |
| 投资方 vs. 公司 + 创始人（连带） | 🟡 公司部分效力存疑，创始人部分通常有效 |

> ⚠️ 建议：将回购义务主要约定为**创始人个人**承担，公司作为补充担保。

## 四、关键术语说明

| 术语 | 中文说明 |
|---|---|
| Participating Preferred | 参与型优先股：清算时先拿回投资本金，再按比例参与剩余分配 |
| Non-participating Preferred | 非参与型优先股：在优先清算和按比例分配中选其一 |
| Full Ratchet | 完全棘轮：下轮融资低于本轮价格时，本轮转换价格调整为下轮价格 |
| Broad-based WA | 宽基加权平均：分母包含所有已发行股份（对创始人友好）|
| Drag-along | 领售权：大股东可强制小股东跟随出售 |
| Tag-along | 随售权：小股东可跟随大股东出售条件 |
| Good/Bad Leaver | 善意/恶意离职：决定离职创始人的股份处理方式 |
| ROFR | 优先购买权：股东转让前须先向其他股东/投资方报价 |

# 劳动合同专项上下文

适用范围：固定期限劳动合同、无固定期限劳动合同、以完成一定工作任务为期限的劳动合同、劳务派遣协议、实习协议（参考适用）。

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **试用期** | 期限、工资标准、考核条件、提前解除程序 |
| **薪酬结构** | 基本工资、绩效工资、补贴、社保基数、发放日 |
| **工时制度** | 标准工时 / 不定时工时 / 综合计算工时（后两者需政府审批） |
| **竞业限制** | 限制范围（地域/行业）、期限、经济补偿标准、违约金 |
| **保密义务** | 保密信息范围、保密期限、脱密期 |
| **知识产权归属** | 职务发明认定、历史 IP 界定、离职后归属 |
| **培训服务期** | 培训内容、服务期期限、违约金上限 |
| **离职条款** | 离职提前通知期、交接义务、离职证明 |
| **违约责任** | 违约金约定（仅竞业限制和服务期两种情形合法）|

## 二、风险评估专项检查清单

### 2-A 用人单位立场

#### 🔴 自动触发高危

1. **知识产权归属不清晰**
   - 未明确约定职务发明归公司所有（《专利法》第六条；《著作权法》第十八条）
   - 未要求员工披露入职前的个人 IP，导致后续权属争议
   - 软件开发岗位未约定"工作成果著作权归公司"

2. **竞业限制条款形同虚设**
   - 竞业限制范围过于宽泛（全行业/全国），司法实践中可能被判定无效
   - 未约定补偿金标准（无补偿则竞业限制对劳动者无约束力，《劳动合同法》第二十三条）
   - 违约金约定过低，无实际威慑力

3. **试用期约定违法**（可能导致支付赔偿金）
   - 试用期超过法定上限（《劳动合同法》第十九条）：3年以上 → 最长6个月；1-3年 → 最长2个月；1年以下 → 最长1个月
   - 以完成一定工作任务为期限的合同不得约定试用期
   - 同一用人单位与同一劳动者只能约定一次试用期

4. **解除条款风险**
   - 未明确"不能胜任工作"的认定标准，导致调岗/解除争议
   - 试用期解除未写明理由，面临违法解除赔偿金风险（2N）

5. **保密条款无实质约束力**
   - 保密信息定义过于宽泛（"所有公司信息均为保密"），法院可能认为不具体
   - 未规定保密期限（建议在职期间 + 离职后 2-3 年）

#### 🟡 中等风险（用人单位视角）

- 股权激励 / 期权协议与劳动合同冲突（需单独处理，不建议写入劳动合同正文）
- 工作地点约定过于固定，影响日后调派灵活性
- 未约定不定时工时或综合工时制，导致加班费风险

---

### 2-B 劳动者立场

#### 🔴 自动触发高危（凡此类条款一律无效，须标注并告知劳动者法定权利）

1. **违法扣押证件或收取押金**（《劳动合同法》第九条明确禁止）
   - 扣押身份证、毕业证等证件
   - 约定入职押金、保证金、"培训费"押金

2. **试用期工资低于转正工资 80%** 或低于当地最低工资（《劳动合同法》第二十条）

3. **约定劳动者支付违法违约金**
   - 仅在竞业限制和专项培训两种情形下，用人单位可约定劳动者承担违约金（《劳动合同法》第二十二条、第二十三条）
   - 其他情形约定劳动者支付违约金一律无效

4. **缺失法定必备条款**（《劳动合同法》第十七条）→ 已在通用合规清单中覆盖

5. **"自愿放弃社保"条款**：无效，社保缴纳是法定义务，不可约定放弃

6. **无固定期限合同应签而未签**
   - 满足以下任一情形应签无固定期限合同（《劳动合同法》第十四条）：
     - 连续工作满十年
     - 连续签订两次固定期限合同后第三次续签
     - 用人单位初次实行劳动合同制度时满十年且距退休不足十年

#### 🟡 中等风险（劳动者视角）

- 竞业限制范围过宽（地域/行业），是否合理？补偿金是否足额？
- 服务期违约金是否与实际培训费用相当（不得超过培训费用剩余部分）
- 工作地点约定不明，可能被单方调派
- 绩效工资占比过高，固定收入保障不足

## 三、合规检查专项清单

依据《劳动合同法》第十七条，以下为法定必备条款（任何遗漏均须标记）：

- [ ] 用人单位名称、住所、法定代表人或主要负责人
- [ ] 劳动者姓名、住址、居民身份证号
- [ ] **劳动合同期限**（固定/无固定/任务制，需明确起止日期）
- [ ] **工作内容和工作地点**
- [ ] **工作时间和休息休假**（标准/不定时/综合工时须标注）
- [ ] **劳动报酬**（基本工资金额、发放方式和日期）
- [ ] **社会保险**（五险：养老、医疗、失业、工伤、生育）
- [ ] **劳动保护、劳动条件和职业危害防护**
- [ ] 法律、法规规定的其他事项

**违法条款黑名单（自动触发无效标注）：**

| 违法约定 | 法律依据 |
|---|---|
| 排除劳动者依法享有的权利 | 《劳动合同法》第二十六条 |
| 用人单位免除自身法定责任 | 《劳动合同法》第二十六条 |
| 约定适用期间不计入劳动合同期限 | 《劳动合同法》第十九条第四款 |
| 收取押金、保证金 | 《劳动合同法》第九条 |
| 扣押劳动者证件 | 《劳动合同法》第九条 |

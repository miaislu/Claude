# 电商平台技术服务协议专项上下文

适用范围：电商平台与商家签订的入驻协议、技术服务协议、平台使用协议、店铺运营协议、平台规则（视为协议）。

> ⚠️ **特别说明**：此类协议通常为平台单方制定的**格式合同**，商家只能接受或拒绝，不能谈判。审查时须重点识别《民法典》第四百九十七条规定的无效格式条款，以及《电子商务法》对平台的强制性约束。

---

## 一、专项条款类别（补充通用分类）

| 专项类别 | 典型条款内容 |
|---|---|
| **入驻条件与资质** | 营业执照、品牌授权、行业资质、保证金 |
| **平台服务费用** | 技术服务费、交易佣金、广告服务费、活动报名费 |
| **保证金机制** | 保证金金额、扣划条件、退还条件与时限 |
| **结算机制** | 结算周期、货款冻结期、扣款项目清单 |
| **店铺管理权** | 平台单方下架、关店、封号的条件与程序 |
| **违规处罚机制** | 扣分规则、处罚金额、申诉程序与期限 |
| **纠纷处理规则** | 退款先行垫付、消费者纠纷裁决规则 |
| **数据权属** | 交易数据归属、商家数据使用权 |
| **排他经营要求** | "二选一"条款（反垄断法高度敏感）|
| **平台规则变更** | 规则单方修改、通知方式、生效时间 |
| **知识产权** | 商家内容授权、平台使用权范围 |

## 二、风险评估专项检查清单

### 2-A 商家（入驻方）立场

#### 🔴 自动触发高危

1. **保证金扣划条件模糊或过于宽泛** ⚡
   - "平台认为有必要时"、"根据平台判断"等单方扣划条款
   - 扣划后无明确的申诉通道和退还时限
   - 保证金退还期限超过 15 个工作日，或退还设置障碍（如需"平台审核通过"）

2. **平台单方关店/封号权无边界** ⚡
   - 平台可基于"违反平台规则"（规则内容未明确）单方关闭店铺，无须提前通知
   - 关店后货款冻结期不明确或无上限
   - 缺少申诉程序和期限（《电子商务法》第三十二条要求平台提供申诉途径）

3. **退款先行垫付责任过重** ⚡
   - 未经商家确认即向消费者退款并从商家账户扣款
   - 纠纷处理规则明显倾向消费者，商家举证责任过重
   - 无上限的垫付义务（如虚假交易、批量刷单退款）

4. **排他经营要求（"二选一"）** ⚡
   - 明确或变相要求商家不得在竞争平台开店或同时促销
   - 依据《反垄断法》第二十二条，具有市场支配地位的平台实施"二选一"构成滥用市场支配地位
   - 即便平台无明显支配地位，此类条款仍存在反垄断风险

5. **平台规则单方修改无实质通知** ⚡
   - 规则修改仅在平台内部页面发布，无邮件/短信通知
   - 修改通知期少于 7 天（《电子商务法》第三十二条要求提前公示）
   - 商家继续使用即视为同意（但未给予实质选择权）

6. **数据权属对商家明显不利**
   - 商家交易数据（含客户数据）归平台所有，商家无任何使用权
   - 平台可将商家数据用于竞争分析、向竞争对手展示

#### 🟡 中等风险（商家视角）

- 结算周期超过 T+15（行业惯例 T+7 至 T+15）
- 活动报名后不可退出，违约金高于活动收益预期
- 平台促销活动为"邀请制"但参与规则不透明
- 知识产权侵权投诉处理机制偏向投诉方（诺提即删，申诉时限过长）

---

### 2-B 平台方立场

#### 🔴 自动触发高危

1. **违规条款违反《电子商务法》强制性要求**
   - 未提供申诉途径（第三十二条）
   - 规则修改未提前公示（第三十二条）
   - 平台内纠纷处理规则不公平、不透明（第六十条）

2. **格式条款存在被认定无效的风险**（《民法典》第四百九十七条）
   - 不合理免除平台责任的条款
   - 加重商家义务且对应权利不足的条款

3. **反垄断合规风险**
   - 排他条款被认定为滥用市场支配地位（《反垄断法》第二十二条）
   - 大数据杀熟条款（差异化定价）可能违反《电子商务法》第十八条

#### 🟡 中等风险（平台视角）

- 保证金扣划依据若不具体，面临商家要求返还的法律风险
- 数据授权范围若无边界，存在被认定侵犯商家商业秘密的风险

## 三、合规检查专项清单

### 《电子商务法》强制要求

- [ ] 平台须明示向经营者收取的平台使用费、交易佣金（第十条）
- [ ] 平台制定的格式合同须公平合理，提供申诉途径（第三十二条）
- [ ] 规则修改须提前公示不少于 7 天（第三十二条）
- [ ] 不得违法要求经营者签订排他协议（第三十五条）
- [ ] 平台内纠纷处理机制须公平、透明（第六十条）
- [ ] 停业关店须提前 7 天通知经营者（第三十七条）

### 《反垄断法》敏感条款核查

- [ ] 是否存在要求商家排他经营的条款（第二十二条）
- [ ] 是否存在差异化定价或大数据杀熟条款（需结合市场支配地位判断）
- [ ] 平台联合经营者固定价格的条款（第十六条）

### 《消费者权益保护法》连带责任核查

- [ ] 平台对商家经营信息的审核义务是否明确（第四十四条）
- [ ] 消费者纠纷中平台的先行赔付义务和追偿机制是否合规

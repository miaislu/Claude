# 诉讼 / 仲裁专项上下文

适用范围：仲裁协议（独立条款或合同附件）、诉讼保全协议（包括诉前保全担保合同）、和解协议（诉讼中和解/庭外和解）、调解协议（人民调解/商事调解/司法确认调解）、仲裁裁决执行协议。

---

## 特别说明：诉讼 / 仲裁语境下的立场判断

诉讼与仲裁文件的审查，与普通商事合同审查存在根本差异：**当事方通常已处于或即将进入对抗性程序**，每一条款都可能直接影响胜诉概率、执行可能性和谈判筹码。

**审查前必须确认的立场信息：**

| 确认项 | 说明 |
|---|---|
| 委托方角色 | 申请方/原告（主张权利方）还是被申请方/被告（防御方）？ |
| 程序阶段 | 纠纷尚未启动 / 已立案 / 已有裁决 / 执行阶段 |
| 争议标的 | 合同纠纷 / 侵权 / 公司纠纷 / 劳动争议 / 其他 |
| 涉外因素 | 是否涉及外国当事人、境外仲裁机构、外国法适用 |
| 诉讼时效状态 | 委托方权利的诉讼时效是否已届满或即将届满 |

> ⚠️ 若委托方角色不明确，**禁止将对方有利条款认定为委托方风险**；应先确认立场再评估。

## 零、商业决策 vs 起草技术——诉讼 / 仲裁文件分类参考

修改建议引擎在处理诉讼 / 仲裁相关文件时，须按下表分类。**分类决定建议格式和读者**：商业决策需当事人拍板，起草技术问题可直接指示律师修改。

### 🏢 商业决策（需当事人 / 决策层拍板）

这类条款反映双方在对抗程序中的策略取舍，没有客观正确答案，决策取决于当事人的风险偏好、证据强弱和谈判地位。

| 条款 | 典型商业决策点 |
|---|---|
| **仲裁机构选择** | CIETAC / ICC / HKIAC / SIAC？国内机构还是境外机构？ |
| **仲裁地点** | 北京 / 上海 / 香港 / 新加坡？影响临时措施可用性和执行便利性 |
| **仲裁员人数** | 独任仲裁员（效率高、成本低）vs. 三人庭（适合大额争议）？ |
| **和解金额** | 愿意接受多少折让换取确定性？ |
| **和解支付方式** | 一次性支付 vs. 分期支付？分期违约后果如何？ |
| **保密条款范围** | 和解金额是否保密？是否允许向监管机构披露？ |
| **免责声明范围** | 互相免责还是单方免责？是否豁免欺诈行为？ |
| **调解协议是否申请司法确认** | 申请确认可获执行力，但增加时间成本 |
| **保全担保金额** | 担保金额高低直接影响保全申请成本 |

### ⚖️ 起草技术（可直接指示律师修改，无需业务决策）

这类问题有客观正确的修改方向，属于起草技术问题。

| 条款 | 典型起草问题 |
|---|---|
| **仲裁条款缺少机构名称** | 仅写"提交仲裁"而未指定机构，导致条款无效或难以执行 |
| **仲裁条款与管辖条款并存** | 同一合同既约定仲裁又约定法院管辖，导致条款冲突 |
| **仲裁条款适用范围表述不清** | "本合同项下争议"vs."与本合同有关的一切争议"，范围差异显著 |
| **和解协议未明确履行期限** | 仅约定"尽快支付"，缺乏具体日期和逾期后果 |
| **和解协议缺少撤诉/撤案条款** | 未约定和解完成后申请撤诉的时限和程序 |
| **调解协议缺少司法确认申请条款** | 当事人不知可申请司法确认赋予执行力 |
| **保全担保金额计算依据不明** | 担保金额与保全标的不匹配，法院可能不予受理 |
| **送达地址条款不完整** | 缺少送达方式、地址变更通知义务，影响程序推进 |
| **仲裁语言未约定** | 涉外仲裁中语言约定缺失，增加程序成本 |

> ⚠️ **分类边界案例**：和解金额折让幅度——到底是商业还是技术？
> - 如果是「双方对折让幅度本身尚未达成共识」→ 🏢 商业决策
> - 如果是「双方已同意八折，但合同写成了按原金额支付」→ ⚖️ 起草技术
> 判断依据：**分歧在于数字本身，还是数字的写法**。

## 一、专项条款类别（补充条款分析师的通用分类）

条款分析师在处理诉讼 / 仲裁相关文件时，须在标准分类之外额外识别以下类别：

| 专项类别 | 典型条款内容 |
|---|---|
| **管辖约定** | 仲裁机构名称、仲裁地、诉讼法院级别与地点、专属管辖排除 |
| **仲裁条款独立性** | 仲裁条款效力独立于主合同、仲裁条款可分离原则 |
| **仲裁庭组成** | 仲裁员人数、仲裁员资格要求、仲裁员指定方式 |
| **仲裁程序规则** | 适用的仲裁规则版本、语言、开庭地点、证据规则 |
| **临时措施 / 保全** | 保全申请权利、保全担保金额与方式、担保解除条件 |
| **诉讼时效约定** | 时效起算点、时效中止 / 中断约定、时效放弃条款 |
| **和解条款** | 和解金额、支付时间表、支付方式、逾期后果 |
| **撤诉 / 撤案安排** | 撤诉时间节点、撤诉条件、未履行和解时的程序恢复 |
| **免责与豁免** | 免责范围、豁免权利的具体列举、保留权利声明 |
| **保密义务** | 保密主体、保密内容（金额 / 事实 / 程序）、保密例外 |
| **执行条款** | 执行地点、执行方式、执行费用负担、协助执行义务 |
| **送达条款** | 送达地址、送达方式、地址变更通知、视为送达条件 |
| **适用法律** | 主合同准据法、仲裁协议准据法（两者可不同）|
| **调解协议确认** | 是否申请人民法院司法确认、申请期限、申请主体 |
| **费用分担** | 仲裁费 / 诉讼费负担方式、律师费是否可主张 |

## 二、风险评估专项检查清单

### 核心法律红线（自动核查，任何违反均高危）

#### 🔴 仲裁条款效力红线

**仲裁条款无效的法定情形（《仲裁法》第十七条）：**

| 无效情形 | 说明 |
|---|---|
| 约定的仲裁事项超出法定仲裁范围 | 婚姻、收养、监护、扶养、继承纠纷及行政争议不可仲裁 |
| 无民事行为能力或限制民事行为能力人订立的仲裁协议 | 主体资格缺陷 |
| 一方以胁迫手段迫使另一方订立的仲裁协议 | 意思表示瑕疵 |

**仲裁条款效力不确定的高危情形：**

- 仲裁机构名称不明确或不存在（如仅写"提交北京仲裁"而无机构全称）
- 同一合同同时约定仲裁和法院诉讼（通常仲裁优先，但存在争议）
- 约定适用某仲裁机构规则但未指定该机构管辖（规则与机构分离）
- 仲裁条款仅约定"友好协商解决"而无明确仲裁意思表示

#### 🔴 诉讼时效红线（《民法典》第一百八十八条）

**一般诉讼时效：3年**，自权利人知道或应当知道权利受损及义务人之日起算。

| 时效类型 | 期限 | 法律依据 |
|---|---|---|
| 一般诉讼时效 | 3年 | 《民法典》第一百八十八条 [本地库] |
| 最长权利保护期间 | 20年（特殊情况可延长）| 《民法典》第一百八十八条第二款 [本地库] |
| 人身损害赔偿 | 3年（按一般时效）| 《民法典》第一百八十八条 [本地库] |
| 国际货物买卖、技术进出口合同 | 4年 | 《民法典》第五百九十四条 [本地库] |

**约定限制诉讼时效的无效情形（《民法典》第一百九十七条）：**
- 约定放弃诉讼时效利益（提前放弃无效）
- 约定诉讼时效期间短于法定期间（缩短无效）
- 约定延长诉讼时效期间（延长有效，但不得违反最长保护期间）

> ⚠️ 合同中常见"本协议项下权利应在 [X 年] 内主张"的约定——若 X 年短于 3 年，该约定无效，仍适用法定 3 年时效。

#### 🔴 保全担保红线

**申请保全须提供担保（《民事诉讼法》第一百零三条）：**

- 诉前保全：须提供担保，否则法院不予保全
- 诉中保全：法院可要求提供担保
- 担保金额：通常为保全标的金额的 20%–30%（各地法院裁量不同）
- 担保方式：现金、银行保函、财产担保均可

**错误保全的赔偿责任（《民事诉讼法》第一百零五条）：**
- 申请有错误，申请人须赔偿被申请人因保全所受损失
- 高危情形：保全标的金额远超实际债权，或明知无胜诉可能仍申请保全

---

### 2-A 申请方 / 原告立场

#### 🔴 自动触发高危的情形

1. **仲裁条款效力存疑**
   - 仲裁机构名称不准确或已更名（如"中国国际商会仲裁院"实为"CIETAC"，需核实）
   - 仲裁条款与诉讼管辖条款并存，对方可能提出管辖权异议拖延程序
   - 仲裁条款仅约定"友好协商"，无明确仲裁意思表示，可能被认定无效

2. **诉讼时效已届满或即将届满**
   - 权利人知道或应当知道权利受损已超过 3 年，且无中止/中断事由
   - 和解协议中约定的"保留追诉权利"条款是否构成时效中断存在争议
   - 建议：在和解协议中明确约定时效中断或重新计算

3. **保全担保安排不足**
   - 担保金额不足，法院可能要求补充担保
   - 担保方式不被法院接受（如部分法院不接受第三方保函）
   - 错误保全风险：若最终败诉，须赔偿被申请方因保全遭受的损失

4. **和解协议缺乏执行保障**
   - 和解协议未经司法确认，仅具合同效力，对方不履行须重新起诉
   - 分期支付安排无违约担保（无保证人、无抵押、无质押）
   - 和解协议撤诉条款设计不当：在款项全部到账前已撤诉，丧失程序保障

5. **执行条款缺失或不可执行**
   - 仲裁裁决执行地在境外，未考虑当地承认与执行程序（《纽约公约》适用性）
   - 法院判决执行地在境外，未考虑双边司法协助条约
   - 被申请方/被告无可供执行财产，未提前申请财产保全

6. **送达条款不完整**
   - 被告地址不明确或已变更，送达困难导致程序延误
   - 涉外仲裁中送达方式不符合仲裁规则要求

#### 🟡 中等风险关注项（申请方 / 原告视角）

- 仲裁费用预估：大额争议仲裁费可能高达标的额的 2%–5%，须提前预算
- 仲裁员指定机制：对方可能指定有利于其立场的仲裁员
- 仲裁裁决的终局性：仲裁裁决一裁终局，无上诉救济，须充分评估证据和法律依据
- 保全期间对方可能申请解除保全，须准备应对材料
- 和解协议保密条款：若对方违反保密义务，损害赔偿金额如何约定

---

### 2-B 被申请方 / 被告立场

#### 🔴 自动触发高危的情形

1. **仲裁条款效力异议丧失**
   - 被申请方已实质参与仲裁程序（如提交答辩、出席开庭），可能被认定放弃管辖权异议
   - 管辖权异议须在答辩期内提出，逾期丧失异议权（《仲裁法》第二十条）
   - 仲裁条款效力异议须向仲裁委提出，不得直接向法院提起确认之诉（须先穷尽仲裁救济）

2. **保全措施导致严重经营损失**
   - 银行账户被冻结，导致无法正常经营
   - 核心资产（设备、存货、股权）被查封，影响生产经营
   - 应对策略：提供等值担保申请解除保全（《民事诉讼法》第一百零四条）

3. **和解协议条款不公平**
   - 和解金额远超合理预期，在诉讼压力下被迫接受
   - 和解协议免责范围不完整，对方保留部分索赔权利
   - 和解协议中的认罪或责任认定条款，可能被用于其他程序

4. **被迫接受不利管辖**
   - 合同约定在对方所在地法院管辖，增加应诉成本
   - 仲裁地在境外，增加参与仲裁的时间和费用成本
   - 专属管辖情形（如不动产纠纷）被错误约定为协议管辖

5. **诉讼时效抗辩丧失**
   - 在诉讼时效已届满后，以书面形式向对方确认债务或作出履行承诺，导致时效重新计算
   - 和解谈判中的让步表态被认定为时效中断行为

6. **仲裁裁决执行阶段的异议**
   - 仲裁裁决存在《民事诉讼法》第二百三十七条规定的不予执行情形（须及时提出）
   - 申请撤销仲裁裁决的期限：自收到裁决书之日起 6 个月内（《仲裁法》第五十九条）

#### 🟡 中等风险关注项（被申请方 / 被告视角）

- 反请求权利：被申请方有权就同一合同关系提出反请求，须在答辩期内提出
- 仲裁庭组成异议：对仲裁员资格或中立性有异议，须及时申请回避
- 执行异议：对执行标的有异议，须在执行阶段及时提出执行异议之诉
- 和解协议签署时机：在仲裁/诉讼开庭后签署和解，可能比开庭前签署更有利
- 担保物的选择：提供替代担保解除保全时，担保物的选择影响经营灵活性

## 三、合规检查专项清单

### 仲裁协议合规

- [ ] 仲裁机构名称完整准确（须为现行有效的仲裁机构全称）
- [ ] 仲裁事项属于可仲裁范围（排除婚姻、继承、行政争议等）
- [ ] 仲裁条款与管辖条款不冲突（同一合同不得同时约定两种争议解决方式）
- [ ] 涉外仲裁：仲裁地约定明确（影响临时措施和撤销裁决的法律适用）
- [ ] 仲裁协议准据法明确（与主合同准据法可不同）
- [ ] 仲裁规则版本明确（适用签署时有效版本还是争议发生时有效版本）

### 诉讼时效合规（《民法典》第一百八十八条、第一百九十七条）

- [ ] 合同中无缩短法定诉讼时效的约定
- [ ] 合同中无预先放弃诉讼时效利益的约定
- [ ] 时效起算点约定清晰（如"自违约行为发生之日起"）
- [ ] 时效中断机制完整（催告函、起诉、仲裁申请均可中断时效）
- [ ] 涉外合同：适用外国法时须核查该国时效规定

### 保全担保合规（《民事诉讼法》第一百零三条至第一百零五条；《保全担保司法解释》）

- [ ] 担保金额与保全标的相匹配（通常不低于保全标的额的 20%）
- [ ] 担保方式被受理法院接受（各地法院对保函接受度不同）
- [ ] 诉前保全：须在 30 日内提起诉讼或申请仲裁，否则法院解除保全
- [ ] 担保解除条件明确（履行完毕 / 提供替代担保 / 申请人撤回）
- [ ] 错误保全赔偿责任条款：申请人须对错误保全导致的损失承担赔偿责任

### 和解协议合规

- [ ] 和解协议须以书面形式订立
- [ ] 和解协议当事人须与原诉讼/仲裁当事人一致（第三人参与须明确其法律地位）
- [ ] 和解协议中的免责范围须明确列举（"豁免一切索赔"的表述过于宽泛，须具体化）
- [ ] 和解协议签署后的撤诉安排须明确（条件撤诉 vs. 无条件撤诉）
- [ ] 涉及国有资产的和解：须经有权机关审批，防止国有资产流失

### 调解协议合规（《人民调解法》；《民事诉讼法》第一百九十五条至第一百九十六条）

- [ ] 人民调解协议：双方可向法院申请司法确认，赋予执行力
- [ ] 司法确认申请期限：调解协议生效之日起 30 日内（《民事诉讼法》第一百九十五条）
- [ ] 司法确认管辖：调解组织所在地基层人民法院
- [ ] 调解协议内容不得违反法律、行政法规强制性规定，不得损害国家利益、社会公共利益和第三人合法权益
- [ ] 商事调解协议：建议申请司法确认或公证赋予执行力，避免对方不履行须重新起诉

### 仲裁裁决执行合规

- [ ] 境内仲裁裁决：向被执行人住所地或财产所在地中级人民法院申请执行

## 四、管辖约定合法性专项分析

### 协议管辖的法定要求（《民事诉讼法》第三十五条）

协议管辖须同时满足：

1. **书面形式**：管辖协议须以书面形式订立
2. **连接点要求**：须选择与争议有实际联系的地点的法院（包括：合同签订地、合同履行地、标的物所在地、被告住所地、原告住所地）
3. **级别管辖不得违反**：协议管辖不得改变级别管辖（如不得约定基层法院管辖本应由中级法院管辖的案件）
4. **专属管辖不得违反**：不动产纠纷、港口作业纠纷、遗产继承纠纷等专属管辖案件不得协议管辖

### 常见管辖约定问题

| 约定方式 | 效力评估 | 风险 |
|---|---|---|
| "向甲方所在地法院起诉" | ✅ 通常有效（若甲方住所地有连接点）| 甲乙方互换后连接点是否仍存在 |
| "向北京市第三中级人民法院起诉" | ⚠️ 效力存疑 | 指定具体法院（而非地区）可能被认定无效 |
| "向合同签订地法院起诉" | ✅ 有效 | 合同签订地须明确，否则难以确定管辖法院 |
| "双方同意提交仲裁，也可向法院起诉" | ❌ 无效 | 仲裁与诉讼并存，仲裁条款通常无效 |
| "向上海市高级人民法院起诉" | ⚠️ 级别管辖问题 | 一般民商事案件不属于高院一审管辖 |
| "向香港法院提起诉讼" | ✅ 原则上有效（涉外合同）| 内地执行香港判决须经认可程序 |

### 仲裁条款独立性原则（《仲裁法》第十九条）

仲裁协议独立存在，合同的变更、解除、终止或无效，不影响仲裁协议的效力。

**实践意义：**
- 主合同被认定无效，仲裁条款仍然有效，仲裁庭仍有管辖权
- 主合同被解除，仲裁条款仍然有效，解除争议仍须通过仲裁解决
- 但仲裁条款本身存在独立的无效事由时，仍可被认定无效

## 五、诉讼时效约定与管理专项

### 时效中断的法定事由（《民法典》第一百九十五条）

以下行为导致诉讼时效中断，中断后时效重新计算：

| 中断事由 | 说明 |
|---|---|
| 权利人向义务人提出履行请求 | 催告函、律师函、微信/邮件催款（须保留证据）|
| 义务人同意履行义务 | 书面或口头（须有证据）承认债务 |
| 权利人提起诉讼或仲裁 | 起诉状递交法院 / 仲裁申请书递交仲裁委 |
| 权利人申请仲裁 | 同上 |
| 权利人申请支付令 | 督促程序 |
| 权利人申请宣告义务人破产 | 破产申请 |
| 权利人申请诉前保全 | 保全申请 |

### 时效中止的法定事由（《民法典》第一百九十四条）

以下情形导致时效中止，中止事由消除后，时效继续计算（仅适用于时效期间最后 6 个月内发生的情形）：

- 不可抗力（自然灾害、战争等）
- 无民事行为能力人或限制民事行为能力人无法定代理人
- 继承开始后未确定继承人或遗产管理人
- 权利人被义务人控制
- 其他导致权利人无法主张权利的障碍

### 合同中时效条款的起草建议

**推荐写法：**
```
本协议项下任何一方的索赔权利，适用中国法律关于诉讼时效的规定。
双方确认，任何一方发出的书面催告通知均构成诉讼时效中断事由。
```

**危险写法（须识别并标记）：**
- "本协议项下权利应在 [1年/2年] 内主张，逾期视为放弃" → 缩短时效，约定无效
- "双方放弃就本协议主张诉讼时效抗辩的权利" → 预先放弃时效利益，约定无效
- "本协议项下权利不受诉讼时效限制" → 违反法律强制性规定，约定无效

## 六、保全担保专项分析

### 保全类型与担保要求

| 保全类型 | 法律依据 | 担保要求 | 申请时机 |
|---|---|---|---|
| 诉前财产保全 | 《民事诉讼法》第一百零三条 | 必须提供担保 | 起诉前 |
| 诉中财产保全 | 《民事诉讼法》第一百零三条 | 法院可要求提供担保 | 诉讼过程中 |
| 诉前行为保全 | 《民事诉讼法》第一百零四条 | 必须提供担保 | 起诉前 |
| 诉中行为保全 | 《民事诉讼法》第一百零四条 | 法院可要求提供担保 | 诉讼过程中 |
| 仲裁保全 | 《民事诉讼法》第一百零三条 | 必须提供担保 | 仲裁申请前或仲裁中 |

### 《保全担保司法解释》关键规则

（最高人民法院《关于人民法院办理财产保全案件若干问题的规定》）

- **担保金额**：法院根据保全标的金额、申请人的申请是否有据等综合确定，通常为保全标的额的 20%–30%
- **担保方式**：现金（最优先被接受）、银行保函、保险公司保单、不动产抵押、有价证券质押
- **担保替换**：被申请人可提供等值担保申请解除保全，法院审查后可解除
- **错误保全赔偿**：申请人申请有错误，须赔偿被申请人因保全遭受的损失（《民事诉讼法》第一百零五条）

### 保全担保协议的核心条款

| 条款 | 审查要点 |
|---|---|
| **担保金额** | 是否与保全标的额相匹配？是否预留错误保全赔偿空间？ |
| **担保期限** | 担保期限须覆盖整个诉讼/仲裁周期（含二审/执行）|
| **担保解除条件** | 须明确：判决/裁决生效且执行完毕 / 申请人自愿撤回 / 被申请人提供替代担保 |
| **担保物变现** | 现金担保：直接划拨；保函：须触发保函条件；抵押物：须经拍卖程序 |
| **反担保** | 被申请人提供替代担保时，须确认替代担保的价值和可执行性 |

# /legal labor — 劳动合同专项审查

> 输出规范、法条引用要求和禁止行为见 `agents/_guidelines.md`。

## 功能描述

针对劳动合同的深度专项审查，重点检查《劳动法》《劳动合同法》的强制性规定，保护劳动者和用人单位的合法权益。

## 用法

```
/legal labor <文件路径>
/legal labor 劳动合同.pdf --party 劳动者
/legal labor 劳动合同.pdf --party 用人单位
```

## 参数

| 参数 | 说明 |
|---|---|
| `--party <劳动者/用人单位>` | 指定代表哪方（默认从劳动者角度保护性审查） |
| `--city <城市>` | 指定工作地城市，以匹配当地最低工资标准、社保基数等 |

## 重点审查项目

### 必备条款核查（《劳动合同法》第十七条）

- [ ] 劳动合同期限（固定期限须明确起止日期）
- [ ] 工作内容和工作地点
- [ ] 工作时间和休息休假（标准工时/不定时工时/综合工时须有审批文件）
- [ ] 劳动报酬（基本工资、绩效、发放日期）
- [ ] 社会保险（五险：养老、医疗、失业、工伤、生育）
- [ ] 劳动保护和职业危害防护

### 常见违法条款检查

| 违法情形 | 法律依据 |
|---|---|
| 试用期超过法定上限 | 《劳动合同法》第十九条（3个月/6个月/1-2年上限） |
| 试用期工资低于 80% | 《劳动合同法》第二十条 |
| 要求劳动者缴纳押金 | 《劳动合同法》第九条（明确禁止） |
| 竞业限制补偿金未约定 | 《劳动合同法》第二十三条（必须有经济补偿） |
| 服务期违约金过高 | 《劳动合同法》第二十二条（仅限专项培训情形） |
| 约定固定期限但实为第三次续签 | 《劳动合同法》第十四条（应签无固定期限） |

### 竞业限制条款专项

如合同含竞业限制条款，额外检查：
- 竞业限制范围是否合理（地域、行业、时间）
- 竞业限制期限不超过 2 年（《劳动合同法》第二十四条）
- 每月经济补偿标准（司法实践：不低于离职前 12 个月平均工资的 30%）
- 违约金是否合理

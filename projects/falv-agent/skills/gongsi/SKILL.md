# /falv gongsi — 公司法事务

> 输出规范、法条引用要求和禁止行为见 `agents/_guidelines.md`。

## 功能描述

处理公司法相关的文件审查和起草，包括股权协议、公司章程、股东会决议等。

## 用法

```
/falv gongsi --type <文件类型> [文件路径]
/falv gongsi --type 股权转让 股权转让协议.pdf
/falv gongsi --type 章程 公司章程.txt
/falv gongsi --type 投资协议 term-sheet.pdf
/falv gongsi --type 股权架构     # 无文件，交互式咨询
```

## 支持的文件类型

| 类型 | 主要审查/起草内容 | 依据法规 |
|---|---|---|
| `股权转让` | 转让程序、优先购买权、工商变更 | 公司法第八十四条 |
| `章程` | 必备条款、表决规则、特殊决议事项 | 公司法第二十五条 |
| `股东协议` | 股东权利义务、反稀释、拖带/随售权 | 公司法 + 民法典 |
| `投资协议` | 估值、优先权、回购条款、对赌安排 | 公司法 + 民法典 |
| `合伙协议` | 合伙人权利义务、退出机制 | 合伙企业法 |
| `股权激励` | 期权池、归属条件、行权价格 | 公司法 + 劳动合同法 |

## 重点检查项

### 股权转让

- 有限公司：其他股东优先购买权通知程序（《公司法》第八十四条）
- 国有股权转让：是否需要审批备案
- 外资股权：商务部/市场监管审查要求
- 工商变更登记时限

### 投资协议 / 对赌条款

- 对赌目标（业绩/上市/时间）是否清晰可量化
- 回购价格计算方式
- 最高人民法院关于对赌协议效力的裁判规则（海富投资案规则）：
  - 与公司对赌（补偿公司利益受损）：有效
  - 与股东对赌：通常有效
  - 对赌触发时对公司资本的影响

### 公司章程

依据《公司法》第二十五条检查必备条款，重点关注：
- 一票否决权事项
- 分红政策
- 股东退出机制
- 竞业限制和忠诚义务

## 输出格式

根据 `--type` 参数输出对应的审查报告或文件草稿。

# /legal risk — 风险条款快速评分

> 输出规范、法条引用要求和禁止行为见 `agents/_guidelines.md`。

## 功能描述

快速扫描合同，聚焦输出高风险条款列表和风险分布热力图，不做完整的全面审查。适合快速筛查是否值得深入审查。

## 用法

```
/legal risk <文件路径或粘贴内容>
/legal risk 合同.txt --top 5      # 只显示前5个最高风险条款
/legal risk 合同.txt --party 乙方  # 重点关注对乙方的风险
```

## 参数

| 参数 | 说明 |
|---|---|
| `--top <N>` | 只显示风险最高的 N 个条款，默认显示全部 |
| `--party <甲/乙>` | 从指定方的角度评估风险 |
| `--threshold <分数>` | 只显示风险分数超过阈值的条款（默认 5）|

## 执行流程

1. 调用 `clause-analyzer` Agent 快速提取条款列表
2. 调用 `risk-assessor` Agent 对所有条款评分
3. 按风险分从高到低排序，输出热力表

## 输出格式

```markdown
## 风险扫描结果

**文件**：xxx.txt  
**扫描条款数**：XX 条  
**风险分布**：🔴 高危 X 条 | 🟡 中等 X 条 | 🟢 低风险 X 条

---

### 风险条款排名

| 排名 | 条款位置 | 风险分 | 风险描述 | 影响方 |
|---|---|---|---|---|
| 1 | 第X条 | 🔴 9/10 | ... | 乙方 |
| 2 | 第X条 | 🟡 6/10 | ... | 双方 |

---
> ⚠️ 本分析仅供参考，不构成正式法律意见。
```

# /legal draft — 法律文件起草

> 输出规范、法条引用要求和禁止行为见 `agents/_guidelines.md`。

## 功能描述

根据用户提供的关键信息，起草符合中国法律规范的合同草稿。先通过交互问答收集必要信息，再生成包含所有法定必备条款的完整文本。

## 用法

```
/legal draft
/legal draft --type 劳动合同
/legal draft --type 技术开发合同 --party 委托方
/legal draft --type NDA --brief
```

## 参数

| 参数 | 说明 |
|---|---|
| `--type <类型>` | 指定合同类型，跳过类型选择步骤 |
| `--party <立场>` | 指定起草立场（偏向哪方），影响条款倾向 |
| `--balanced` | 起草平衡版本，不偏向任何一方 |
| `--brief` | 跳过详细问答，仅收集最基本信息后起草 |

---

## 执行流程

### Step 0：类型与立场确认

如未通过参数指定，先询问：

```
您想起草哪类合同？

  1. 劳动合同          6. 商业租赁合同
  2. 技术开发/软件合同  7. 借款合同
  3. 保密协议（NDA）    8. 知识产权许可合同
  4. 服务合同          9. 经销/代理合同
  5. 采购合同         10. 其他（请描述）
```

确认类型后询问立场：

```
这份合同从哪方角度起草？

  A. [甲方/委托方] — 条款偏向保护您的权益
  B. [乙方/服务方] — 条款偏向保护对方权益
  C. 平衡版本       — 双方权益均衡
```

### Step 1：信息收集

按对应类型的"必问清单"逐项收集。

- 标有 `*` 的为必填项，缺失则无法生成合规合同，**必须向用户追问**
- 标有 `?` 的为选填项，用户跳过则以 `【待填写：XXX】` 占位
- `--brief` 模式只问必填项

### Step 2：起草合同

按类型标准结构起草，要求：
1. 包含该类型**所有法定必备条款**（见各类型说明）
2. 基于收集到的信息填充具体内容
3. 未收集的信息用 `【待填写：XXX】` 标注
4. 委托方视角：验收权、解除权、赔偿权条款措辞严格；相对方视角反之

### Step 3：起草后提示

完成后提示：

```
草稿已完成。建议：
  → 运行 /legal review 对此草稿进行全面审查
  → 如需调整，直接告诉我修改哪条
```

## 各类型起草指南

### 1. 劳动合同

**必问清单：**
- `*` 用人单位名称、统一社会信用代码、地址
- `*` 劳动者姓名、身份证号
- `*` 合同类型（固定期限 / 无固定期限 / 任务制）
- `*` 工作内容（岗位名称）、工作地点
- `*` 合同期限（起止日期）、试用期（如有）
- `*` 基本月薪（税前）、发薪日
- `*` 工时制度（标准 / 不定时 / 综合计算）
- `?` 绩效奖金结构
- `?` 竞业限制（范围、期限、每月补偿标准）
- `?` 特殊保密要求

**条款结构（依据《劳动合同法》第17条）：**
> 双方信息 → 合同期限 → 工作内容与地点 → 工作时间与休假 → 劳动报酬（明细） → 社会保险 → 劳动保护 → 试用期 → 保密与竞业 → 违约责任 → 解除与终止 → 争议解决

**立场倾向：**
- 用人单位：强化 IP 归属、竞业限制、服务期条款
- 劳动者：明确工资总额构成、加班费标准、离职补偿权利

---

### 2. 技术开发 / 软件定制合同

**必问清单：**
- `*` 双方基本信息
- `*` 开发内容概述（系统名称、核心功能）
- `*` 合同总价及付款节点（里程碑/验收后）
- `*` 主要交付物（源代码 / 文档 / 测试报告）
- `*` 预计完成时间
- `*` 知识产权归属（归甲方 / 归乙方 / 共有）
- `?` 验收标准（功能列表 / 测试用例）
- `?` 保修期及响应标准
- `?` 是否需要保密条款

**条款结构：**
> 双方信息 → 开发内容与范围 → 里程碑与交付物 → 验收标准与程序（含逾期视为通过） → 合同价款与支付 → 知识产权归属（含背景技术保留） → 保密 → 需求变更程序 → 保修期与缺陷修复 → 违约责任（含责任上限） → 争议解决

**立场倾向：**
- 委托方：强化验收权、IP 归委托方、源代码实时托管
- 开发方：明确变更追费机制、验收逾期视为通过、责任上限 = 合同总价

---

### 3. 保密协议（NDA）

**必问清单：**
- `*` 双方基本信息
- `*` NDA 类型（单向 / 双向）
- `*` 保密信息的范围描述
- `*` 保密期限（在职期间 + 离职后年限）
- `?` 违约金金额
- `?` 排他谈判期（如有）

**条款结构：**
> 保密信息定义（正向 + 排除情形） → 保密义务（禁止披露、禁止使用） → 例外情形 → 保密期限 → 信息归还/销毁 → 违约责任 → 适用法律与争议解决

---

### 4. 服务合同

**必问清单：**
- `*` 双方基本信息
- `*` 服务内容描述
- `*` 服务期限
- `*` 服务费金额及支付方式
- `?` 服务标准 / KPI
- `?` 知识产权归属
- `?` 保密需求

**条款结构：**
> 服务内容与范围 → 服务期限 → 服务费与支付条件 → 服务标准与验收 → 双方权利义务 → 知识产权 → 保密 → 违约责任 → 争议解决

---

### 5. 采购合同

**必问清单：**
- `*` 买卖双方基本信息
- `*` 货物名称、规格型号、数量、单价、总价
- `*` 交货时间与地点
- `*` 付款方式与账期
- `?` 质量标准（国标 / 企标 / 样品确认）
- `?` 验收期限与异议程序
- `?` 质量保证期

**条款结构：**
> 货物描述（含规格） → 数量与价格 → 交货条款 → 验收标准与程序 → 付款条款 → 质量保证 → 违约责任 → 不可抗力 → 争议解决

---

### 6. 商业租赁合同

**必问清单：**
- `*` 房东、租客基本信息
- `*` 租赁物地址、面积（计量方式：建筑面积 / 使用面积）、用途
- `*` 租期（起止日期）、免租期（如有，须说明性质）
- `*` 月租金、付款方式（月付 / 季付 / 年付）
- `*` 押金金额（押几个月）
- `?` 物业费、停车费归属
- `?` 是否允许转租
- `?` 装修权利与限制

**条款结构：**
> 租赁物描述（地址、面积、现状确认） → 租期（含免租期性质） → 租金与支付（含递增） → 押金（扣划条件、退还时限） → 使用与维护责任 → 装修权利 → 转租限制 → 提前退租与违约金 → 续租优先权 → 合同终止与房屋交还 → 争议解决

**立场倾向：**
- 租客：免租期明确为合同权利、提前退租违约金上限 3 个月、押金 30 天内退还
- 房东：连续欠租 15 天触发解除权、装修须书面审批、转租须书面同意

---

### 7. 借款合同

**必问清单：**
- `*` 出借人、借款人基本信息
- `*` 借款金额
- `*` 年利率（**必须确认不超过当前 LPR × 4 倍**）
- `*` 借款期限与还款日期
- `*` 还款方式（到期一次性 / 分期 / 等额本息）
- `?` 担保方式（无 / 保证 / 抵押 / 质押，须明确担保人/物）
- `?` 提前还款条件与违约金

**条款结构：**
> 借款金额与币种 → 借款用途 → 利率（年化，注明基准日 LPR） → 还款计划（含分期明细） → 担保条款（如有） → 提前还款 → 违约事件与罚息 → 陈述与保证 → 争议解决

> ⚠️ 起草完成后自动提示：请在签署前确认年利率不超过 LPR × 4 倍（当前约 14%）。

---

### 8. 知识产权许可合同

**必问清单：**
- `*` 许可方、被许可方基本信息
- `*` 许可的 IP（商标注册号 / 专利号 / 作品名称）
- `*` 许可类型（独占 / 排他 / 普通）
- `*` 许可地域范围
- `*` 许可期限
- `*` 使用费（一次性 / Royalty 比例 / 混合）
- `?` 是否允许转授权
- `?` 改良成果归属

**条款结构：**
> 许可 IP 描述（含权属证明编号） → 许可类型与范围 → 许可期限与续期 → 使用费与支付 → 质量控制（商标必须有） → 改良成果归属 → 转授权限制 → IP 有效性保证 → 维权义务 → 保密（技术秘密） → 合同终止与停用 → 备案登记要求 → 争议解决

---

### 9. 经销 / 代理合同

**必问清单：**
- `*` 品牌方、经销商基本信息
- `*` 合同性质（经销 / 代理，明确区别）
- `*` 授权产品 / 品牌
- `*` 独家还是非独家（如独家，明确区域 / 渠道）
- `*` 合同期限
- `?` 年度最低采购 / 销售指标及未完成后果
- `?` 库存回购条款
- `?` 终止补偿机制

**条款结构：**
> 合同性质确认 → 授权范围（产品/地域/渠道） → 独家权利（如有）及维持条件 → 价格政策（建议零售价，**禁止固定转售价格**） → 最低指标义务 → 市场推广义务 → 库存与备货 → IP 使用授权 → 合同终止（通知期、补偿、库存回购） → 反垄断合规声明 → 争议解决

> ⚠️ 起草完成后自动提示：请确认价格条款仅为建议零售价，不含最低转售价格约定（违反《反垄断法》第18条）。

## 通用起草原则

1. **法条引用**：引用任何法条前先在内部验证存在性，禁止引用已废止法律（见 `_guidelines.md`）
2. **占位符**：未收集的信息用 `【待填写：XXX】` 标注，不留空白
3. **完整性**：每份合同必须包含争议解决条款和不可抗力条款
4. **免责声明**：每份草稿末尾必须附：

> ⚠️ **起草说明**：本草稿由 AI 辅助生成，仅供参考，不构成正式法律意见。标注 `【待填写】` 的内容需双方协商确定。正式签署前，建议交由执业律师审核。

# /legal plain-language — 法律术语白话翻译

> 输出规范、法条引用要求和禁止行为见 `agents/_guidelines.md`。

## 功能描述

将晦涩的法律条款和法律术语翻译为普通人能理解的白话文，同时保留原始法律含义不失真。

## 用法

```
/legal plain-language <文件路径或粘贴文本>
/legal plain-language "不可抗力系指不能预见、不能避免且不能克服的客观情况"
/legal plain-language 合同第五条.txt --level 小学生    # 用最简单的语言解释
/legal plain-language 合同.pdf --glossary              # 同时输出术语表
```

## 参数

| 参数 | 说明 |
|---|---|
| `--level <受众>` | 目标受众：`普通人`（默认）/`商务人士`/`小学生` |
| `--glossary` | 同时输出文中出现的法律术语词汇表 |
| `--keep-original` | 保留原文，翻译内容显示在原文下方 |

## 执行流程

1. 识别文本中的法律术语、专业表达、被动语态合同句式
2. 逐段翻译为白话文
3. 翻译原则：
   - 保持法律含义准确，不过度简化导致误解
   - 用"你""我"代替"甲方""乙方"（当 `--party` 参数指定时）
   - 对重要含义用 **加粗** 标注
4. 在必要时添加注释，说明某条款对当事人的实际影响

## 常见翻译示例

| 法律表达 | 白话翻译 |
|---|---|
| "甲方有权单方解除本合同" | **甲方（合同另一方）可以随时终止这个合同，不需要你同意** |
| "违约方应承担相应的违约责任" | 如果你违反了合同，你需要赔偿对方的损失 |
| "本合同自双方签字盖章之日起生效" | 双方都签了名（加盖公章）之后，合同才正式生效 |
| "不可抗力" | 天灾、战争、政府禁令等你无法控制也无法预防的突发情况 |
| "争议提交仲裁解决" | 如果发生纠纷，不去法院打官司，而是找专门的仲裁机构裁决 |

## 输出格式

```markdown
## 白话翻译结果

---

**原文**：[原始条款]

**白话**：[翻译后的通俗表达]

💡 **实际影响**：[该条款对你意味着什么]

---
```

（如有 `--glossary` 参数，在末尾附加术语表）

# 策略标题映射规则（通用品类）

---

## 一、通用策略标题体系（适用所有品类）

| 策略标题 | 适用举措关键词 |
|---------|-------------|
| **品商策略** | 商品/品类/价格/补贴/货盘/结构/心智品 |
| **场域策略** | 频道/入口/流量/转化/CXR/链路/橱窗/样式 |
| **用户策略** | 新客/老客/红包/分层/CAC/潜新/摇摆用户 |
| **供给策略** | 门店/库存/冷链/配送/开仓/覆盖/闪电仓 |
| **治理策略** | 风险/合规/品质/售后/治理/水位/假货/防伪 |
| **品类运营策略（XX）** | 特定细分品类专项 |
| **营销规划** | 未来活动/节点筹备/规划类 |

---

```
举措关键词

    ↓ 否

```

## 三、品类专项映射示例

### 示例1：酒水品类

#### 啤酒（WBR 2.3）
| 举措 | 策略标题 |
|------|---------|
| 酒类专项券 | **品商策略** |
| 精酿品类运营/周五小酒馆/精酿节 | **品类运营策略（精酿）** |
| 啤酒冰镇覆盖/冷链建设 | **供给策略** |
| 新客获取/AI选品/用户分层 | **用户策略** |
| 歪马精酿选品对齐 | **品类运营策略（精酿）**（标注：歪马协同） |

#### 白酒（WBR 2.4）
| 举措 | 策略标题 |
|------|---------|
| 核心品补贴/白酒投券 | **品商策略** |
| 名酒行/货盘扩充 | **品类运营策略（名酒行）** |
| 防伪码/鉴真/假货治理/风险水位 | **治理策略** |
| 餐渠场景联动/邀酒节 | **营销规划** |

## 四、合并规则

**同类型策略必须合并为一条**，不拆分写多条。

❌ 不合格：
```
**精酿策略：** 推进精酿品类运营
**歪马协同策略：** 与歪马对齐精酿选品
```

✅ 合格：
```
**品类运营策略（精酿）：** ①推进精酿品类日常运营，精酿 GTV WoW+X%；
②与歪马对齐选品池，补充长尾 SKU X 个；效果：精酿新客数 WoW+X%。
```

## 五、外部事件处理

**节日/天气/竞对变化 ≠ 策略举措**，归入「规模」段归因，不写入策略段。
